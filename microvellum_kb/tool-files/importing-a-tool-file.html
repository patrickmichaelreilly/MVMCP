Importing a Tool File

This article explains the steps to import a new toolfile.

  1. Hover over "Toolbox Setup" and select "Options."


  1. Select the "Toolfiles" tab and select the "Import" button.


  1. Select the desired toolfile to import. 
  2. After successful import, observe the toolfile will appear in the toolfile list. 
    • Importing toolfiles can also be achieved by using the  Import/Export utility .
    • For information assigning the toolfile to a processing station,  click here .
