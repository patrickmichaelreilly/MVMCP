Saving a Cabinet to Your Library

After a product has been created or modified, it can be saved to the library to be available for immediate use at any time.
  1. Select the Modify tab.

    2. Fig. 01 – Modify Tab

  2. Select Modify Products.

    3. Fig. 02 – Modify Products

  3. Select Save Products to Library.

    4. Fig. 03 – Save Product to Library

  4. Select the Product to save.
  5. Select the Category in which to save the Product.

    6. Fig. 04 – Save Product to Library Window

  6. After selecting a category, and before selecting Save, enter a name for the Product.

    7. Fig. 05 – Naming New Product

  7. Select Save > Select Yes.


The desired library product will now become usable within the specified category. For information about advanced library edits, see the Database Explorer.
