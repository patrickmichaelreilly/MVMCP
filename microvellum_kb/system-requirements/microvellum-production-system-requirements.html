Recommended System Configuration for Microvellum Production

These system requirements include recommendations for Microvellum Production

Operating System (64 bit): 

  1. Windows 11
  2. Windows 10
    3. Windows 32-bit is not supported.
Processor:
Minimum: 2.5-2.9 gigahertz (GHz) or faster processor
Preferred: 3.0+ gigahertz (GHz) or faster processor
Memory:
Minimum: 8 GB RAM
Preferred: 16 GB RAM
Display Resolution:
Minimum:  1920x1080 with True Color, and 125% Desktop Scaling (120 DPI) or less recommended
Resolutions up to 3840 x 2160 supported on Windows 10, 64-bit systems (with capable display card)
Graphics Card:
Minimum: Will run on most standard Graphics Cards.
Disk Space:
Minimum: 10+ GB SSD or HDD
Preferred: 100+ GB SSD
