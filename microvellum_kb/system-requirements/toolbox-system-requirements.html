Recommended System Configuration for Microvellum Toolbox

These system requirements include recommendations for Toolbox BSB and Toolbox AC (AutoCAD Companion)

Operating System: 

  1. Windows 10
  2. Windows 11
    3. Windows 32-bit is not supported.
Processor:
Minimum: 3+ GHz CPU or faster processor. For example, the Intel® Core™ i5 or equivalent
Recommended: 5+ GHz CPU or faster processor. For example, the Intel® Core™ i7 or equivalent.
Memory:
Minimum: 16 GB RAM
Recommended: 32 GB+ RAM
Display Resolution:
Minimum:  1920x1080 with True Color, and 100% Desktop Scaling (120 DPI) or less recommended
Preferred:  Dual Display 1920x1080 with True Color, and 100% Desktop Scaling (120 DPI) or less recommended
Resolutions up to 3840 x 2160 supported on Windows 10, 64-bit systems (with capable display card)
Graphics Card:
Minimum: 4 GB GPU with 29 GB/s Bandwidth and DirectX 11 compliant with 106 GB/s Bandwidth and DirectX 12 compliant
 
Recommended: 8 GB+ GPU with 106 GB/S Bandwidth and DirectX12 compliant
DirectX 12 with Feature Level 12_0 is required for Shaded (Fast) and Shaded with edges (Fast) visual styles.

AutoCAD uses your computer's display card for a variety of essential graphics operations including but not limited to view manipulation, line smoothing, and text/line type generation.  It is recommended that you have a display card with dedicated VRAM to support these operations at optimal speeds. Visit the Autodesk website for a list of AutoCAD Certified Graphics Hardware.
Disk Space:
  1. Minimum: 20 GB Installation
  2. Recommended: As needed

Additional Requirements for Large Datasets, Point Clouds, 3D Modeling, and Model Documentation:
  1. 16 GB RAM or more
  2. 6.0 GB free hard disk available not including installation requirements
  3. 3840 x 2160 (4K) or greater True Color video display adapter; 4G VRAM or greater; Pixel Shader 3.0 or greater; DirectX-capable workstation class graphics card.
