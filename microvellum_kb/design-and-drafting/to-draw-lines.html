To Draw Lines

  1. Click Home tab > Draw panel > Line.
  2. Specify the start pint and end point of the line segment by clicking in the drawing area. 
  3. Continue specifying additional line segments. 
    4. To undo the previous line segment, enter  u at the prompt. Click Undo on the Quick Access toolbar to cancel the entire series of line segments: 

  4. Press Enter or Esc when done or enter c to close a series of line segments. 

Using Specific Coordinates

  1. Click Home tab > Draw panel > Line. 
  2. Type the coordinate value for the first point by typing the X value, a comma, then the Y value, for example 1.65,4.25. 
  3. Press the Spacebar or Enter. 
  4. Do one of the following:
    1. If dynamic input is on: Type the pound sign (#) followed by the X-value, a comma, then the Y-value, for example #4.0,6.75. 
    2. If dynamic input is off: Type the X value, a comma, then the Y value, for example 4.0,6.75.
  5. Press the Spacebar or Enter. 
When dynamic input is on, relative coordinates are the default. When dynamic input is off, absolute coordinates are the default. Press F12 to turn dynamic input on or off. 

Using a Relative Coordinate

  1. Click Home tab > Draw panel > Line. 
  2. Specify the first point. 
  3. To specify the second point relative to the first point, do one of the following: 
    1. If dynamic input is on: Type the X-value, a comma, then the Y-value, for example 4.0,6.75. 
    2. If dynamic input is off: Type the at sign (@) followed by the X-value, a comma, then the Y-value, for example @4.0,6.75. 
  4. Press the Spacebar or Enter. 

A Specific Length

  1. Click Home tab > Draw panel > Line. 
  2. Specify the start point. 
  3. Do one of the following to specify the length: 
    1. Move the cursor to indicate the direction and angle, and enter the length, for example 6.5. 
    2. Enter the at symbol (@) and the length, followed by the left angle bracket (<) and the angle, for example @6.5<45. 
  4. Press the Spacebar or Enter. 

At a Specific Angle 

  1. Click Home tab > Draw panel > Line. 
  2. Specify the start point. 
  3. Do one of the following to specify the angle: 
    1. Enter the left angle bracket (<) and the angle, for example <45, and move the cursor to indicate the direction. 
    2. Enter polar coordinates, for example 2.5<45. 
    3. Press F8 to turn on Ortho to lock the angle to horizontal and vertical directions. You can also press Shift when specifying the next point for a horizontal or vertical direction. 
    4. Press F10 to turn on polar tracking. You might need to use the DSETTINGS command, Polar Tracking tab to specify additional polar angles and choose the option to track all polar angle settings. 
    5. Move the cursor to indicate an approximate angle. 
  4. Do one of the following to specify the length: 
    1. Click a point to specify the endpoint with or without using object snaps. Trim or extend the resulting line as needed. 
    2. Enter the length of the line, for example 2.5. 
  5. Press the Spacebar or Enter. 

At a Specific Angle to Another Line

Temporarily aligning the UCS icon with an existing line makes it easy to draw a line at a specific angle to another line. 
  1. At the command prompt, enter UCS. 
  2. Enter ob for Object and select the existing line. 
    3. The UCS origin (0,0,0) is redefined. 

  3. Click Home tab > Draw panel > Line. 
  4. Specify the start point. 
  5. Do one of the following to specify the angle: 
    1. Enter the left angle bracket (<) and the angle, for example <45, and move the cursor to indicate the direction. 
    2. Move the cursor to indicate the approximate angle. 
  6. Specify the second point. 
  7. Press the Spacebar or Enter. 
  8. At the command prompt, enter UCS. 
  9. Enter p for Previous to reset the UCS origin.
