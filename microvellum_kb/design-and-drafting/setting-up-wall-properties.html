How to Set Up Wall Properties

Wall Properties

To access the Walls Setup, click on the Toolbox Options menu and select Walls Setup. 

The Wall Setup interface will allow you to set defaults for when you draw walls. These settings can be changed per project or room. 

   
  1. Default Wall Thickness – Use this setting to fix the wall thickness when drawing walls.
  2. Default Wall Height – Use this setting to set the wall height when drawing walls.
  3. Wall View Oversize Dimension – This setting is used for creating an oversized viewport if using AutoCAD’s Sheet Sets. Enter a value greater than 0 to activate.
  4. Add “VERIFY” to Dimensions – Use this setting to add “VERIFY” to wall dimensions.
  5. Dimension Elevation Length – Use this setting to add a dimension to the length of the wall. The Distance from Wall setting is relative to the vertical placement of the dimension above the wall.
  6. Dimension Elevation Height – Use this setting to add a dimension to the height of the wall. The Distance from Wall setting is relative to the horizontal placement of the dimension beside the wall.
  7. Label Elevation – Use this setting to insert an elevation reference token (block) for 2D elevated walls. The Distance from Wall setting is relative to the vertical placement of the block below the wall. The Wall Label Drawing Name is used to specify which drawing block to use. Scale Wall Label from Current Scale is helpful when using a standard drawing template (non-annotative).
