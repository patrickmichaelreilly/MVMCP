How to Modify Multiple Products using Edit Multiple Product Prompts

Multiple products can be modified using the Edit Multiple Product Prompts function. You can access the prompts for a product by right-clicking in the Draw Products window, or by selecting the Modify tab in Toolbox and selecting the Edit Multiple Product Prompts under the Modify Products menu.  


  1. Prompt Selection  – Allows you to select which prompts to be displayed in the product table.
  2. Product Table  – Displays the products that were selected in the drawing and displays some default prompts as well as the prompts that were selected from the prompts list.
  3. Quick Access Menu  – By right-clicking on a product, this menu will display the prompts available for editing. You can also open the Product Prompts for individual products.
  4. Edit Selected Product Rows Only  – Allows you to edit the prompts for the selected product(s) only if checked. If unchecked, all the products in the list will receive the same prompt changes.
  5. Edit Saved Column List  – Allows you to add/remove prompts from the prompt list to/from the product table. Any saved prompts will be displayed in the future.
  6. Redraw   Products on Close  – Checking this box will cause the products in the product table to be redrawn upon closing the interface.
