Modifying Walls using the Prompts Interface

You can access the prompts, for a wall, by right-clicking in the Draw Room Components window. You can also select the Modify tab in Toolbox and select Properties under the Modify Room Components menu (or you can access them by clicking on the Product Prompts icon  if you are using the tool-strip). 

With the prompts interface open, you can modify the properties of the selected wall. You can add obstructions like a column or bulkhead (furrout or furrdown). You can add windows, doors, and openings.  


Although you can adjust the width of the wall from this interface, it is recommended to use the Adjust Wall Length command to do so. The Adjust Wall Length command gives you the added ability to adjust the widths of products associated to the selected wall.
