Understanding Child Bands in Report Designer

The articles in this section of the Knowledge Base involve working with Child Bands for display in your reports. To access the Report Designer, select Toolbox Setup > Design Reports > Design (Report Type) Report. Select New Report to design a report from scratch or New Report from Existing. Select Designer to open Report Designer.

From this screen, add a Child Band component to your report, or edit an existing child band component.


Fig. 01 – Child Bands

A Child band can be placed after any other band on a page. Working in tandem with the “Parent” band, the Child band essentially extends the Parent band. Still, it can have different formatting applied to it, like a different background color, for example.

Using the Child Band with Data Bands

Using the Child band allows you to print two bands on one data row. Create a report, or edit an existing, insert a Data band, and insert a Child band after the Data band.


Fig. 02 – Child Band Inserted Below Data Band

Running the report results in the Child band being printed as many times as the Data band. The example below demonstrates how the Child band is a continuation of the Data band but still possesses its own properties.
