Foundation Library Subassembly Inserts

This article will provide you with an overview of the Subassembly Inserts included within Microvellum's Foundation Library. Each subassembly has its own set of configurable options and can be inserted into a variety of Foundation Library products, including products from the Closet/Wardrobe Library Expansion.

Internal Inserts

Shelf Adj

  

Description: Adjustable Shelf Subassembly Internal Insert designed to be added into any product’s square opening.  This subassembly is included with the base Foundation Library.

Main Options:

·         Parametric Sizing Width/Height/Depth

·         Shelf Qty to specify the number of shelves

·         Shelf Height Calculator

·         Shelf Setback variable to push shelves in from the opening face

·         Continuous hole pattern or individual line bores per shelf

·         Shelf Standards option

·         Shelf edge options for Edgebanding or Lip Up / Lip Down

·         Optional parametric shaped edge detail

·         Shelf hole variables for locations & spacings

·         Access to Material Options

Additional Info: The insertion point for this subassembly insert is the lower left-hand corner at the front. The width/height/depth are defining the available area that the shelves can be located.

Division With Adj Shelves

 

Description: Single Division with Adjustable Shelves Subassembly Internal Insert designed to be added into any product’s square opening.  This subassembly is included with the base Foundation Library.

Main Options:

·         Parametric Sizing Width/Height/Depth

·         Shelf Qty to specify the number of shelves

·         Division Width Calculator to specify the two vertical opening sizes

·         Division Setback variable to push division & shelves in from opening face

·         Double Division option

·         Full Top Shelf above Division option

·         Use the Rail Notch option if you require the divisions to create a locating notch in the rails above

·         Specify Shelves Left, Right, or Both

·         Shelf edge options for Edgebanding or Lip Up / Lip Down*

·         Shelf hole variables for locations & spacings*

·         Access to All Construction Methods

·         Access to Material Options

Additional Info: The insertion point for this subassembly insert is the lower left-hand corner at the front. The width/height/depth are defining the available area that the parts are located. *Adjustable shelves originate from the Shelf Adj insert (nested subassembly).

Pigeon Holes

Hanging Rail, Double Hanging Rail

 

Description:  Hanging Rail Subassembly Internal Insert designed to be added into any product’s square opening.  This subassembly is included with the base Foundation Library.

Main Options:

·         Parametric Sizing Width/Height/Depth

·         Hanging Rail Qty up to four

·         Rail location/position variables

·         Side Clearance variable to deduct from the rail for the support hardware

·         Options to change Hanging Rail Type & Support hardware

·         Top fixed shelf option to align above the rail

·         Option to display 3D clothes hangers

·         Access to All Construction Methods

·         Access to Material Options

Additional Info: The insertion point for this subassembly insert is the lower left-hand corner at the front. The width/height/depth are defining the available area that the parts are located.  

 

Horizontal Opening Splitter Component

 

Description:  Horizontal Opening Splitter Subassembly Internal Insert designed to be added into any product’s square opening.  The intent of this insert is to horizontally “split” any opening with vertical divisions. This subassembly is included with the base Foundation Library.

Main Options:

·         Parametric Sizing Width/Height/Depth

·         Division Width Calculator to specify the opening sizes & locations

·         Double Division option

·         Component Setback variable to push parts in from opening face

·         Full Top Shelf above Division option

Additional Info:  Each bay has access to six Foundation Library Component Inserts (Shelf ADJ, Shelf Fixed, Divisions, Pigeon Holes, Wine Rack Trellis, or Hanging Rail).

The insertion point for this subassembly insert is the lower left-hand corner at the front. The width/height/depth are defining the available area that the parts are located.  

 

Vertical Opening Splitter Component

 

Description:  Vertical Opening Splitter Subassembly Internal Insert designed to be added into any product’s square opening.  The intent of this insert is to vertically “split” any opening with horizontal fixed shelves. This subassembly is included with the base Foundation Library.

·         Vertical Bay Calculator to specify the opening sizes & locations

The insertion point for this subassembly insert is the lower left-hand corner at the front. The width/height/depth are defining the available area that the parts are located.
