Overview: Foundation Library Handles and Pull Tabs

The video below covers working with Handles and Pull Tabs within the Microvellum Foundation Library.

How to View Handles/Pulls

To view the available handles and pulls in your Library:

  1. Open Toolbox.
  2. Go to the Toolbox Setup menu and choose Library Specification Groups.
  3. In the Specification Group interface, select the Material File.
  4. In the Material File, select the Hardware Library tab.
  5. In the Hardware Library, navigate to the Handles/Pulls section to view the available options.


How to Change Handles/Pulls at the Project Level

  1. Go to Toolbox Setup and choose Project Specification Group.
  2. In the Specification Group interface, select the Material File Wizard and then choose the Hardware Wizard from the drop-down menu.
  3. In the Hardware Wizard, navigate to the Handles/Pulls tab.
  4. To change the handle or pull, click the button next to the text box and select a new option from your library. The new hardware will be applied at the project level. Products will need to be redrawn for changes to be reflected in the drawing.
