Overview: Crown Molding

The video below walks you through the process of adding Crown Molding to your project from the Microvellum Foundation Library. It also goes into further detail about how to extend Crown Moldings for various cabinet layouts.

How to Change Default Crown Molding

  1. Open the Toolbox Setup and choose Project Specification Groups from the drop-down menu.
  2.  In the list of specification groups, select the Global Variables for your project.
  3.  In the General Settings tab, navigate to the Moldings section.
  4. Under Crown Moldings, select the desired default crown molding.


How to Add a New Crown Molding

  1. With the Global Variables interface open, select View in the top left and Show LookUp Tables from the drop-down menu. 
  2. In the bottom left of the interface, select the “LookUpTableMoldingProfiles” from the Table Name drop-down menu. 
  3. Insert a new row in desired location of new molding and fill out the Name, Profile File Name, Picture File Name, Height, Depth, and Y_Offset of the new Crown Molding. 


To Extend Crown Molding Without Any Visible Joins

1. Make sure that you have selected the option to add Crown Molding to your desired product in the "Construction Options" section in the Product Prompts.


2. Go to "Modify Subassemblies" and choose "Show Subassembly Prompts." Then, select the crown molding subassembly.



3. Check the boxes labeled "Return Left" and "Return Right," which will allow you to extend the crown molding in either direction.

4. Double Click in the text box to measure the depth of the adjacent cabinets on the side where you want to extend the crown molding. (either "Adjacent Cab Depth for Left" or "Adjacent Cab Depth for Right"). For example, in the drawing below I will be measuring the depth of the Upper Cabinets on the left of the Tall Cabinet.



5. To determine the length of the extension, measure the distance from the end of the crown molding to the point where you want it to extend to by double clicking in the Extend Crown Molding text box.



6. Check the box labeled "Return Extended Crown Molding Left/Right," depending on which direction you want to extend the crown molding.

7. Make sure that the miter angles for the crown molding are set to 45 degrees or your desired angle.

8. Click "OK" to apply the changes and redraw the product.
