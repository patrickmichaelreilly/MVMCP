Issue: SQL Connection Error - Temporarily Change Database Type in the Registry

Some Microvellum users may get an error related to SQL server connection issues such as: "A connection to the factory database could not be established. Please select a different configuration."

Symptom: Toolbox starts up but freezes, and you're unable to click Start Toolbox.

  • If this is your issue, be sure to close Toolbox via Task Manager before making changes to the registry.

These steps tell Toolbox to check your local database so that you can log in and reconnect to the SQL server.

  • Left-click on "Start Menu" and type "regedit.exe."

  • Follow the path as shown for your version:

    • OEM 2016:

    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R7.5\Toolbox-F001:409 \Configurations\[Configuration Name]\Database Settings

    OEM 2017:

    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R7.5\Toolbox-0001:409 \Configurations\[Configuration Name]\Database Settings

    OEM 2018:

    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R18.0\Toolbox-1001:409 \Configurations\[Configuration Name]\Database Settings

    OEM 2019:
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R19.0\Toolbox-2001:409 \Configurations\[Configuration Name]\Database Settings
    OEM 2020:
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R20.0\Toolbox-3101:409\Configurations\[Configuration Name]\Database Settings
    OEM 2021:
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R21.0\Toolbox-4101:409\Configurations\[Configuration Name]\Database Settings
    OEM 2022:
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R22.0\Toolbox-5101:409\Configurations\[Configuration Name]\Database Settings
    OEM 2023:
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R23.0\Toolbox-6101:409\Configurations\[Configuration Name]\Database Settings
    OEM 2024:
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R24.0\Toolbox-7101:409\Configurations\[Configuration Name]\Database Settings
    OEM 2025: 
    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R25.0\Toolbox-8101:409\Configurations\[Configuration Name]\Database Settings

    Toolbox Standard:

    HKEY_CURRENT_USER\Software\Microvellum\Toolbox\R7.0\Toolbox-Standard \Configurations\[Configuration Name]\Database Settings

    Micromanager/ERP: 

    HKEY_CURRENT_USER\Software\Microvellum\MicroManager\R7.0\MicroManager \Configurations \[Configuration Name]\Database Settings

    ALIS (Legacy):

    HKEY_CURRENT_USER\Software\Microvellum\ALIS\R7.0\ALIS \Configurations\[Configuration Name]\Database Settings

  • Right-click on "DataBaseType" and "Modify" – change "SQL" to "SQLCE" – restart the program.

  • Startup Toolbox, go to Toolbox Setup > Options > Data Access > Set to SQL Server and reconnect to your SQL server by DataCatalog and DataSource -- Be sure to test the connection!
