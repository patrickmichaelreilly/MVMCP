Issue: Error Message "Toolbox VBA is not Currently Installed"

Issue:

While using Toolbox v7, you'll come across the following error:
"Toolbox VBA is not currently installed.  Please run the Toolbox R7 installer to install the Toolbox VBA component." 

Solution:

Toolbox v7 no longer uses any VBA components, but a few commands may still check to see if that is installed.  This error should only occur one-time and then never display again; however, if it does continue to occur, you can resolve the issue by extracting the attached file to the following directory:
C:\Program Files\Common Files\Microsoft Shared\Office 10
