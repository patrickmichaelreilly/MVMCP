Announcing a New Edgebanding Token

A new edgebanding token has been developed that expands the capabilities of the types of part edges for which Toolbox calculates lengths. This new EDGEBAND token allows you to add additional edgebanding to the edge of a part that contains ‘non-original edges.’ These are edges that are defined by route machining that modify the original rectangular part edges. For example, they may be border routes that are added to a part that creates a corner notch cutout as for a toe kick notch. It may also be a multi-arced route that is added to a part that creates a wavy edge to the part.

Use the new token to assign new edgebanding to a part with one of the two different methods reviewed in this article How to Use the Edgebanding Token. In addition to the new token, an option has been added to the Single Part Editor that will allow you to create an edgebanding token from a newly created polyline route. 

The new Edgebanding Token is available in Toolbox 2020 | Build 20.1.0214 or higher.
