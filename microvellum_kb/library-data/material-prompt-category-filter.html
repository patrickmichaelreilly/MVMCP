Material Prompt Category Filter

A Material Prompts Category Filter can be applied to prompt control types 7, 8, and 9.  In the prompt tab of Edit Design Data and Library Designer, the Combo Box List column has been repurposed and renamed Control Type Data.  For combo box control types, the Control Type Data column will still be used for Combo Box List data.  However, for prompt control types 7, 8, and 9, Control Type Data can now be used to add a filter to material categories in the material selection UI. 

Enter the keywords for the filter, separated by commas, into the Control Data Type column.  Each keyword is added to an include list, meaning that the category must contain that keyword in order to be displayed.  If the keyword starts with a minus sign, it is added to the exclude list, meaning the category will not be displayed if it contains that keyword.  All parent and child categories of a filtered category will be displayed.  If a child category contains an excluded keyword it will not be displayed in the tree view. 

For instance, say you wanted to show all Pulls except for the Glass subcategory.  You would enter "Pulls, -Glass" into the Control Type Data column of the prompt.   The interface also contains a toggle button to turn off the filter and display all categories in the tree view.
