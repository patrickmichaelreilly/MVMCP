Overview: Invisible Fasteners Hole Pattern Custom Spacing

Using the library build 048.4 it is now possible to configure invisible fasteners such as Lock Dowel on a custom 32mm spacing pattern, or another spacing pattern as desired, instead of only the auto space equal pattern.

To activate this capability, open the global variables and set the following variable.

“Global Variables > Machining > General Machining > Invisible fasteners > Lockdowel Channel Lock > Invisible Fasteners Hole Pattern > Custom Space”


Fig. 1


Open the G! workbook in Workbook Designer and scroll to column AD, row 2040 for the lookup table. 


Fig. 2


Column AD contains various part sizes.

Column AE contains the 32mm spacing for the incremental part sizes. The larger the panel grows, the more 32mm holes are added. These spacing values may be modified if your company has differing requirements. You may add rows if more spacing combinations are required.

Columns AE and AF contain data for different types of invisible fasteners (e.g., Lockdowel Channel Lock, Lockdowel H-Clip).

Typically, smaller parts such as nailers, stretchers, and toe-kicks contain only one or two fasteners. To accommodate these smaller parts, the main Invisible Fasteners lookup table contains a column (3 – Horiz Hole Start Offset (Small)) that specifies the offset distance from the edge of the part. The formulas in this column may be modified to 32mm or 64mm as desired to obtain the desired start distance. This column is used instead of the custom spaced holes table described above.


Fig. 3


Setting the spacing in the lookup table and redrawing the product results in spacing that conforms to the appropriate Lockdowel standard.
