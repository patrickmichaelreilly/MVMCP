Understanding the Edit Design Data Interface

The Edit Design Data interface is a spreadsheet editor, similar to Microsoft Excel. 


The interface includes tools located at the top of the window. Below the tool icons is a drop-down list of defined names that can be used to locate the cell’s reference. Beside the drop-down is the formula bar. Formulas will be displayed and can be edited using the formula bar. Below the formula bar is the spreadsheet. At the bottom of the spreadsheet, area are the tabs that the product contains. At the bottom of the interface are different save and close options.

The tools available (from left to right) include:
  1. Save – Allows you to save the changes made to the spreadsheet.
  2. Close Subassembly – Allows you to close out of a subassembly or nested subassembly.
    3. This icon is only visible when a subassembly or nested subassembly file is open. When a nested subassembly is open, the icon will have a green arrow.
  3. Cut – Allows you to cut a cell, or range of cells, from the spreadsheet.
  4. Copy – Allows you to copy a cell, or range of cells, from the spreadsheet.
  5. Paste – Allows you to paste a cut or copied cell, or range of cells, to another location.
  6. Insert Function – Allows you to access the Formula Editor, which can be used to read or write formulas.
  7. Force Recalculation – Allows you to force formulas and equations to recalculate.
  8. Undo – Allows you to undo changes.
  9. Redo – Allows you to redo changes that were undone.
  10. Find and Replace – Allows you to find every mention of the keyword entered, and replace every instance. 
  11. Name Manager – Opens the Name Manager window. 
  12. Filter Parts – Allows you to view included parts (parts with a quantity greater than 0) or all parts.
  13. Toggle Part Thickness Column – Allows you to hide or view the thickness column.
  14. Toggle Hidden Columns A through P – Hides or Displays hidden columns. 
  15. Show Spreadsheet Errors – Allows you to find any errors within the selected spreadsheet tab.
  16. Set Standard Column Widths – Allows you to set standard widths for the columns within the selected spreadsheet.
  17. Workbook Designer – Allows you to access the Workbook Designer interface. Workbook designer includes more spreadsheet tools and allows you to access the other workbooks within the project.

A noticeable feature that is built into Edit Design Data is the conditional formatting to make cells with formulas appear with red text. Black text indicates a static value.
