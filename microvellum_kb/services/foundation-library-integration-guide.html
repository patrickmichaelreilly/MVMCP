Integration Guide for the Foundation Library

This article contains an attached PDF of Microvellum's Foundation Library Integration Guide, a document which explains how to set up the Microvellum Foundation Library and the various different variables, products, and types of data available to users. Form fields are pre-filled with either Metric or Imperial library default values, depending on which version is downloaded.  
This integration guide contains details and settings for Microvellum’s Foundation Library - Build 24.1127.
