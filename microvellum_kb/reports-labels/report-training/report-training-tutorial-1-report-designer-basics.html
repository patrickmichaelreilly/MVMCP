Report Training Tutorial 1: Report Designer Basics

This is the first of a series of five Reports Training Tutorial videos.  This video presents some basic concepts that will be useful for first-time users of Report Designer.  By the end of this video, you will have some familiarity with the user interface and common report configuration tools.  You'll learn how to customize your work environment and set up toolbars and quick launch buttons as you create a custom roll feed label and convert it into a sheet label report.
