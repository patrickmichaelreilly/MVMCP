Configure Activity Stations for Project Costing

In this article, we will show you how to configure your Activity Stations to match the default configuration of the Cost Based Estimating Report.    

As pointed out in the article How to Use Project Costing Tools, if you modify your activity stations and those modifications include modifying the activity station numbers, you also must modify the costing reports with your revised numbers.
Registered users may access downloads of the most current Reports and Activity Stations in the Downloads section of the Microvellum Knowledge Base.
We will demonstrate this procedure based on the assumption that you have not altered your activity station numbers. If for some reason, you need to modify those numbers, we highly recommend that you obtain the services of a Microvellum report developer to modify your reports to match.
  1. Open Toolbox and click Toolbox Setup > Options > Processing Stations.
  2. Click the Activity Stations button. It is the button with the two small circles and opposing arced blue arrows. Since the Activity Stations are set up for each separate configuration, it doesn’t matter which Processing Station is selected.

Fig. 1 - Accessing the Activity Stations

  1. In the section ’01 Cutting of Parts’, click the station named ‘101 – Part Cutting.’ The property descriptions are as follows, accompanied by the values found in the Cabinet Component Library v050.5.
    1. Activity Station Name: “101 – Part Cutting”
    2. Activity Station Number: 101 (use the default numbers for the stations as found in the Help Center article How to Use the Microvellum Project Costing Tools Figure 5 as the default activity station numbers are hard-coded strings in the costing reports.)
    3. Unit Type: Lineal Feet
    4. Units Per Minute of Factory Time: 15
    5. Number of Employees Required: 1
    6. Labor Rate: 250
    7. Comments: “All sawing or border routing of parts.”
    8. Parent Station: [Enter a parent station if the station you’re modifying is dependent on, or a secondary operation to another station. This property does not apply to the majority of Activity Stations.]
    9. Update button: [Click this button to save the properties of a single activity station before moving on to the properties for another station without having to close the interface.]
    10. Available for These Shifts:
      1. Primary Shift: (not checked)
      2. Second Shift: (not checked)
      3. Saturday Shift: (not checked)
  2. After setting the properties for the first station, click the button ‘Update’ and select the next station to edit from the list on the left.
  3. When all the station properties have been set, click OK.

Fig. 2 - Activity Station Properties


Once you have completed the steps above for all the activity stations in a configuration, you are ready to create work order data and view the Cost Based Esitmating Report.
