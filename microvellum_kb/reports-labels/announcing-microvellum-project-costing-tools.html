Announcing Microvellum Project Costing Tools

We are pleased to announce the new Microvellum Project Costing Tools in Toolbox. This article presents an overall view of this new Costing capability.


Previous versions of estimating and costing relied on library data configurations and those legacy tools continue to be supported in the new version of the program. However, this new method of costing does not require extensive library data prompt modifications and specific programs such as the Production module to produce estimates. It contains just three significant components that are needed to activate the full range of costing in Toolbox.

  1. Material file material cost amounts and/or labor time values for each material.
  2. Activity stations labor time and labor rate for each station.
  3. The Cost Based Estimating Report.

The Project Costing Tools are available with the Activity Stations configured in the Component Cabinet Library v050.5 or greater. If you have a library older than that version you may add the appropriate Activity Stations or modify the reports to match your existing stations. You must use Toolbox builds 15.6 or greater but these tools are not dependent on a specific program version.


See How to Use the Toolbox Project Costing Tools in the Microvellum Knowledge Base for a reference article explaining details of the Project Costing Tools.

See Configure Materials for Project Costing in the Microvellum Knowledge Base for a tutorial showing how to set up your material files for the Project Costing Tools.

See Configure Activity Stations for Project Costing in the Microvellum Knowledge Base for a tutorial showing how to set up your activity stations for the Project Costing Tools.

See Print Reports for Project Costing in the Microvellum Knowledge Base for a tutorial showing how to set up your activity stations for the Project Costing Tools.

Registered users may access downloads of the most current Reports and Activity Stations in the Downloads section of the Microvellum Knowledge Base.
