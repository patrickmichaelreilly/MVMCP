Style Designer

Access the Style Designer by selecting the Style Designer icon in the Home tab of the ribbon.


Fig. 1 – Style Designer Access

The Style Designer  allows for the creation and editing of styles in reports. Styles and their collections impact the appearance of reports.

See Figure 2 for a breakdown of the Style Designer Dialog.


Fig. 2 – Style Designer Interface

  1. The Toolbar features commands used in Style Designer.
  2. The Primary pane features a list of styles and collections.
  3. The properties panel, or the “Object Inspector,” of the selected style.
  4. This panel describes the currently selected property.
  5. The Formatting panel is for styles. It contains the standard controls for formatting components.

Toolbar


Fig. 3 – Style Designer Toolbar

  1. The Actions menu contains basic commands to control styles and style collections:
    1. Open – Through this selection, the user can open a previously saved file with a list of styles and style collections.
    2. Save As – Used for saving style and style collections to the *.sts file.
    3. Create Style Collection – Used to create an automatic collection of styles.

      4. Fig. 4 – Actions Menu

  2. The Add Style menu features the styles that are available to be added.

    3. Fig. 5 – List of Available Styles

  3. Used for deleting a style or collection from the list.
  4. Select a component in a report > Open the Style Designer > Select this button. The style of the selected component will be copied.
  5. Used for filtering and sorting.

    6. Fig. 6 – Filter Parameters

  6. The search field is helpful for quickly locating a style.
