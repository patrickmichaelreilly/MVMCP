How to Run an Update for Toolbox

This article covers the steps involved in updating your Microvellum Toolbox Software Installation.  Before updating, we recommend reviewing our Pre-Installation Instructions and creating a backup of your data.  For information on creating a backup see the Knowledge Base article Microvellum Data Backup Tips.

When planning for an update it’s important to keep in mind that anyone using the software will need to learn and adapt to new or updated software features.  Learn about the changes to the software by reviewing the Release Notes for all builds since your last update.  Consider how these software changes will affect your processes and workflow and plan accordingly.

Best practice: Create a sandbox environment to test new software updates before updating your production workstations.  For more information, check out the Microvellum Sandbox Environment Guide.

Updating Toolbox

  1. Right-click on the Toolbox update installation file and select “Run as administrator."

  2.  Begin the installation by selecting “Install.”

  3. A success notification will appear when the installation is complete.  Click "Finish."

  4. You have successfully updated your Microvellum software.
