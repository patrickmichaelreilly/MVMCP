Microvellum Foundation Library Release Notes | Build 23.1010

It is heavily recommended that users downloading this Foundation Library build update their Toolbox software to build 23.1.1010.641 or newer, as this update contains features that rely on improvements made in Toolbox build 23.1.1010.641. 

Additions

  1. Added King Slide SIMLEAD metal drawer system & ULEAD Series undermount drawer runners.
  2. Added additional Lamello Fasteners:
  1. Bisco P-10 & P-15
  2. Tenso P-10 & P-14
  1. Added a Miter option for “Fixed and Adj shelf Construction Method Shelf Edges”.
  2. Added the ability to remove the rear stop dado. Added two new global variables “Dado Notch Distance to Rear Edge” and “Dado Small Part Notch Distance To Rear Edge” and linked to all relevant parts with DADO tokens. 
  3. Added missing global variable “Lock Drop Front Distance”. Wood Drawers will now reference this instead of a fixed value inside the subassembly.
  4. Added Post Laminate feature to all cabinet products. Added new “Post Laminate” subassembly that activates when “G!Post_Laminate=1” and a face is selected as finished (Left, Right, Top, Bottom, Back). Global variables “Post Laminate” and “Plam Overhang” should now be visible in the interface.
  5. Added the ability to rebate (rabbet) cabinet sides for backs with clearance holes at each corner. “G!Rebate_Lead” will need to be adjusted (0.5mm). Added two new global variables:
    1. “Rebate Back Clearance Holes”
    2. “Rebate Back Clearance Hole Diameter”
  6. Added content to support a new Bathroom Partition expansion. 53 new global variables, 7 new material pointers, 5 new edge banding pointers, Metlam hardware material library, 12 profile drawings, and 1 new factory lookuptable. Also added new category of hardware materials. Please contact your account manager for more information about adding this expansion.

Modifications

  1. Updated Richelieu Drawer Slide .dwg graphics by reducing their file sizes, lessening performance impact when using.
  2. Drawer Division Machine Point has been changed to force machining to face 5.

Resolutions

General Resolutions

  1. Resolved an issue that occurred with Base 1 Drawer Microwave when set to Full Inset Door.
  2. Resolved an issue with Master Double Sided Cabinet. Rear bay components will no longer reference front bay depth.
  3. Resolved an issue with Master Under Stair Cabinet when using two doors.
  4. Resolved several issues with the “Hardware Array” construction feature.
    1. Divisions now use the same start/stop variables as Top/Bottom parts. All parts should now reference the hardware array table.
    2. Top Stretchers & Mid Stretchers will now use the Hardware_Array_Small_Part_Threshold to correct their pattern layouts.
    3. Mid Stretchers will now have hardware on the same face left/right.
    4. Hanging Rail Shelves now include Hardware Array tokens.
  5. Resolved a Lock Location issue at Door Drawer Cabinets when base locks are set to lock to top.
  6. Resolved a Lock Location issue at Upper Cabinets when lock location is set to “Lock to Bottom” and a bottom valance is used.
  7. Resolved a few items inside the ADA Sink Angled Frame product. The side routes will no longer reference the horizontal drawer gap.
  8. Resolved a sink cutout issue inside the Countertop with Substrate product.
  9. Resolved a hinge plate location issue with Door Drawer Blind Corner Cabinet(s) when using Use Blind Corner Hinge, Add Blind Corner Panel, and Overlay Blind Corner Panel.
  10. Resolved an issue with all toe-kick box assembly subassemblies and products, wherein the sub front did not account for any values placed into the Sub Front Height prompt.
  11. Resolved an issue with Upper Undermount Rangehood Cabinet when set to Full Capture back.
  12. Resolved an issue with rear internal inserts involving the line bore holes not shifting in relation of the cabinet modifiers.
  13. Master Double Sided Cabinet will now have Pullout Options front/back.
  14. Resolved an issue with cabinets using overlay top, dado construction, and top finished. Applying edge banding to cabinets with these settings turned on would lead to errors. If the settings are applied, edge banding will no longer apply to these products.
  15. Resolved an issue with MV Profile Door. The 3D drawing only tokens have been updated to properly represent what will be machined (using current MV versions).
  16. All Base and Tall Corner Cabinet toe-kicks will now have 2D plan drawings. Also, 2D elevation grain indicators and finish labels corrected.
  17. Resolved issues with the Elias Woodwork Doors’ center panel location. Elias Door wizard has been updated and included with the new .exe library install only (not included in the library update).
  18. Resolved issues with Titus Tekform Drawers. 113 height rear back brackets added and resolved issue with Back Over Bottom option.
  19. Resolved an issue with the Titus Tekform Slimline Drawer's back.
  20. Resolved an issue with Grass Nova Pro Scala that occurred when using Rear Cutout option.
  21. Resolved an issue with Blum Legrabox Inner Drawer (NZ region) that occurred when using K & C heights (graphics added).
  22. Resolved an issue with Krome Premium Drawers front drilling.
  23. Resolved an issue with the Wood Drawer Box when using a rear cutout.
  24. Resolved an issue in Costing Reports related to Drafting Labor.

Face Frame Cabinetry Resolutions

  1. Resolved an issue found in Vertical Face Frame cabinets, wherein the Fixed Shelf FF could collide with the side when using applied ends.
  2. Removed edge banding from certain drawer stretchers & fixed shelves when behind a face frame.

Office Furniture Resolution

  1. Resolved the issue of regular backs not rebating into sides.
