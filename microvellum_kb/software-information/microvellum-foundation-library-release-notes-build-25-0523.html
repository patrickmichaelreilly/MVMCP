Microvellum Foundation Library Release Notes | Build 25.0523

Additions

  1. Added a new Master Curved Front Cabinet product.
  2. Added 282 Titus Hinges and 83 Titus Base Plates.
NOTE: You will need to manually add this hardware brand into Hardware Wizard “Hinge Brand” combo box list if you are updating library.
    1. T-Type 95° Thick Door
    2. T-Type 110°
    3. T-Type 110° Pie Cut Corner
    4. T-Type 110°/15°
    5. T-Type 110°/30°
    6. T-Type 110°/45°
    7. T-Type 110°/90° Blind Corner
    8. T-Type 120°
    9. T-Type 170°
    10. T-Type Mounting Plates
  1. Added new hinge hardware, the Blum CLIP top 120° hinges.
  2. Added a new global variable “Include Reveals In Hinge Offset From Edge”. Select this to include the door reveals when calculating the hinge cup offset from the edge of the door for full overlay conditions. Variable is located under the Hardware tab, Hinge Settings.
  3. Added a new global variable, “Back Capture at FE”. Enable to always have the cabinet backs captured at a Finished End. Disable to prevent the backs from being captured at Finished Ends.
  4. Added a new option, "Pullout Behind Door", to the “Component Option” prompt.
    1. When "Pullout" is selected, the face panel behaves like a drawer front—no hinges are applied.
    2. When "Pullout Behind Door" is selected, the face functions as a standard hinged door.
    3. The "Horizontal Handles Centered" prompt has been added to all blind corner products. Selecting "Pullout" now automatically applies horizontal handles and centered handles.
  5. Added the new Component Pullout subassembly insert. This can be used to add a pullout to any opening with a finished face attached.
  6. “Mullion” prompts are now visible and placed on the Construction Options tab for all 3 door cabinets and blind corner cabinets, as well as the fixed and adjustable shelves. A prompt has been added to control the width of the cutout depth and the clearance: Mullion_Width_Clearance and Mullion_Width.
  7. Added the “Shelf Qty Left” and “Shelf Qty Right” prompt variables to the Division With Adj Shelves and Division with Fixed Shelves subassemblies. This will make it easier to have different shelf quantities in each opening.
  8. Added the “Magic Wire” feature to all subassemblies with Fixed Shelves and Divisions. For Divisions, an option was included for continuous holes to be added tops & bottoms.
  9. Added the new Wire Management hardware: Cabiseal Pestblock (available in Australia). This can be best utilized with the Universal Hardware Insert subassembly.
  10. Added the Blum Tip-On feature to the Wood Drawer Box & Wood Dovetail Drawer Box subassemblies. 
  11. Added two new hardware wizard prompts to separate the cabinet Screw Fitting pointer from the screws used on a wood drawer box and wood drawer box bottom:
    1. “Screw Fitting Wood Drawer Box”
    2. “Screw Fitting Wood Drawer Box Bottom”
  12. Added A & M ADA Hardware Brackets to the ADA Sink Angled Frame product.
  13. Added a new global and door variable, "Vertical Rail To Hinge Minimum Offset". This has been added to improve control when the top hinge needs to be offset to accommodate a vertical front rail. Previously, the top hinge offset was calculated as the sum of Vertical_Front_Rail_Width + G!Hinge_Location_Top, which often resulted in an offset not required for some vertically aligned doors. This has been refined—now the hinge is offset only by Vertical_Front_Rail_Width + Vertical_Rail_To_Hinge_Minimum_Offset, providing a more accurate adjustment.
  14. Added new options to the Suspension Brackets subassembly. The left and right suspension brackets can now be independently toggled on or off.
  15. The Wall Rail product has been updated to support different profiles and multiple length calculations:
    1. The rail part now references a lookup table to apply the appropriate DWG profile dynamically.
    2. A thickness column has been added to the lookup table to apply the correct thickness of the profile.
    3. The calculation logic for wall rail lengths has been improved to dynamically determine the required quantity and sizing based on the product width and available profile stock lengths.
  16. Added the “Plan Width Base Point” prompt to all relevant products and linked to the global variable. This allows local control at the cabinet level to position the 2D plan width dimension None, Back, or Front.
  17. Added two new door summary and fabrication reports that sorts by the subassembly and door style, showing total quantities:
    1. Door And Drawer Front Summary Report
    2. Door And Drawer Front Fabrication Report Style 2
  18. Added new set of global prompts to provide greater control over which doors receive a lock in a double-door cabinet:
    1. “Lock Side Double Door Base”
    2. “Lock Side Double Door Upper”
    3. “Lock Side Double Door Tall”
  19. The below global variables have been moved to the “Legacy” tab due to no longer being used:
    1. “Lock Left Door Base”
    2. “Lock Left Door Tall”
    3. “Lock Left Door Upper”
  20. When “Both Doors” is selected for “Lock Side Double Door Base”, the lock position will automatically change to "Lock To Top". For “Lock Side Double Door Upper”, the lock position will automatically change to "Lock To Bottom". For “Lock Side Double Door Tall”, it will stay looking at the global “Lock Tall Location”. For all options, the Lock Location can still be manually changed in the door subassembly.

Modifications/Resolutions General

  1. Updated the Under Panel product. Strip light parts have been removed and replaced with the Undermount Strip Lighting Insert subassembly, which has more features and is easier to maintain.
  2. Resolved an issue with Fillers in suspended cabinets.
  3. Resolved an issue with right hand Return Fillers option.
  4. Resolved issues with Component Drawers subassembly. Using “Add Top Facia” will now function properly and the drawer stretcher for the facia will now be placed in the correct X location.
  5. Cabinet Connector Holes will no longer automatically appear on cabinet sides with a filler.
  6. Resolved a route priority machining issue with the Blum Aventos HKi lift-up hardware.
  7. Updates have been applied to all shipping DWT template files. Outdated font types have been removed.
  8. Updated the Hinge BLUM 170deg Full Overlay Knock-In hardware to include machine tokens for machining.
  9. Updated the Fixed Shelves w Dividers subassembly to include the proper number of dividers with 10 fixed shelves.
  10. Several reported issues with the Blum SPACE STEP toe-kick drawer subassembly.
  11. Updated the Wood Drawer Box & Wood Dovetail Drawer Box so all “File Drawer” prompts are now visible and more accurate. Added File Bar Clip graphics. Added a prompt to activate the machined slot.
  12. Resolved an issue with Farmhouse Sink Single Bowl type. Sides will now be the same thickness.
  13. Resolved an issue with the Mitered Shelf product. When using the global “Miter on CNC” option, routes will now be the full depth of the part instead of the miterfold depth.
  14. All system drawer subassemblies have been updated to work with the reduced back and bottom rebates (routes) when using materials thicker than 16mm. Many internal formulas have been reformatted for better consistency and accuracy involving the drawer backs and their 3D positioning. 
  15. Resolved issues with the Wood Dovetail Drawer Box when using the Rear Cutout feature.
  16. Resolved an issue with the Fixed Panel option in Master Horizontal cabinets.
  17. Resolved the missing graphics for the Titus Tekform Slimline when used as an inner drawer.
  18. The LEGRABOX Kit has been removed from the hardware list of the Gollinuci Evo 580 bin subassembly.
  19. Drawer subassemblies’ prompts “Bottom Fixing Screw Type” and “Back Fixing Screw Type” now point to H!Default_Drawer_Wood_Screw instead of H!Default_Drawer_System_Screw.
  20. The following hardware wizard variables have been moved from the “Additional Hardware” tab to the “Drawers” tab:*
    1. "Dowel Fitting Wood Drawers"
    2. "Default Drawer System Screw"
    3. "Default Drawer Wood Screw"
    4. "Hardware Edge Array Wood Drawers"
  21. Resolved an issue in the MV Profile Door subassembly. The “Panel Insert Clearance” prompt will now correctly reference D!Panel_Insert_Clearance.
  22. Resolved an issue with Solid Wood Doors when using Glass Lites openings. Mullion parts will now have the correct Y origin location when door is a nested subassembly.
  23. Resolved an issue in the 2D Instructions subassembly that caused issues with the 2D plan view showing “FE” text when it shouldn’t.
  24. Resolved an issue with Top Scribe/Filler Panels on cabinets not correctly perfect grain matching.
  25. Resolved an issue with right hinge plates on Angled End Cabinet and Transition Cabinets using “Angle Right”. Plates will now properly associate to the right sides.

Additions/Resolutions Closet/Wardrobe Expansion

  1. Resolved machining issues with the Kessebohmer Conero Clothes Lift TOP and Kessebohmer Conero Clothes Lift subassemblies.
  2. Hanging Notch and Wall Hanger options have been integrated across all closet products for improved installation flexibility and configurability.
  3. Closet Carcase and Corner Closet Carcase subassemblies have been updated to include hanging notch features.
  4. The Suspension Brackets subassembly is now nested within to support wall hanger hardware.
  5. The following global variables have been introduced to manage these enhancements:
    1. G!Closet_Hanger_Notch_Type – Selects the type of hanging notch required to create a notch at the back of a closets, allowing a hanging cleat to run through. None, Standard, or French Cleat
    2. G!Closet_Hanger_Notch_Distance_From_Top – Sets the vertical position of the notch from the top.
    3. G!Closet_Hanger_Notch_Width – Controls the width of the hanging notch.
    4. G!Closet_Hanger_Notch_Depth – Controls the depth of the hanging notch.
    5. G!Closet_Add_Wall_Hangers – Toggles the inclusion of suspension brackets.
  6. A new Closet category has been added to globals: Closet Hanger Notch Settings.
  7. Created a Closet Wall Cleat product and included it a new closet expansion category: Closet Accessories.
  8. Updated the hanging notch tokens in closet products to accommodate a French Cleat cutout option.

Additions/Resolutions Face Frame Cabinetry

  1. Resolved an issue with “Separate Bead” option when used with the global “Face Frame Extend Bottom Rail to Floor”.
  2. Resolved an issue with the global "Face Frame Miter FE Joint” when used in conjunction with cabinet Applied End feature.
  3. Resolved a major issue with Master Upper Split Height Designer Cabinet FF that was caused from a typo inside a formula.
  4. Resolved an issue with Transition Angle Cabinet FF when side condition is set to Face Frame. Applied FF panels will now miter correctly and respond to "G!Face_Frame_Miter_FE_Joint".

Modifications/Resolutions Staircase Expansion

  1. Made several changes to enable 3D visualization of the nosing radius for imperial:
    1. Added global LookUpTable: LookupTableStaircaseNosing
    2. The Global Prompt “Staircase_Nosing_Radius” is now linked to the lookup table for the available radii and has been changed to a dropdown list. 
    3. Nosing on the treads have been changed from a Profile token to a polyline token. The polyline will now look at the lookup table for the tool to be used and if it’s to be machined or drawing only.
    4. The Global "Staircase_Stringer_Finish_Tool" prompt is now linked to the lookup table. This is the route that is used for the finish cut on the stringer notching rebate.

*Not included in Library Update, only in the full install.
