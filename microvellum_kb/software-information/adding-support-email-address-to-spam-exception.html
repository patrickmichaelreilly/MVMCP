Adding Microvellum Support Email Address to Spam Exception

This tutorial demonstrates the steps recommended for adding the Microvellum Support email address to your spam exception list in order to ensure you are staying updated and in touch with the Microvellum Support team.

Gmail 

  1. Log into Gmail. 
  2. Select “Settings” > “See all settings” located in the top right corner. 
  3. Select “Filters and Blocked Addresses” > “Create a new filter”. 
  4. In the “From” selection, input support@microvellum.zohodesk.com
  5. Select “Create Filter”. 
  6. Select the options “Never send it to Spam”, “Always mark it as important”, and “Categorize as: Primary”. 
  7. Select ‘Create Filter”. 

Outlook 

  1. Log into Outlook online 
  2. Select “Settings” > “View all Outlook settings” located in the top right corner. 
  3. Select “Mail” > “Junk email”, scroll down to the section “Safe senders and domains”. 
  4. Select “Add” and input the email support@microvellum.zohodesk.com and hit enter.
