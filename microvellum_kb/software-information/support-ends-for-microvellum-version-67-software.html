Support Discontinued for Microvellum Version 6.7 Software

As of December 1, 2021, all forms of support for Microvellum version 6.7 software applications will be discontinued. If you are still using Microvellum v6.7 software, then you know that maintenance updates ended in 2012. This announcement pertains to the discontinuation of support activities related to troubleshooting, configuration, and general technical support assistance for Microvellum v6.7 software. 

Why Is Support Being Discontinued?

As mentioned above, maintenance updates were discontinued nine years ago. This means  this outdated software was not developed to support current versions of AutoCAD or Windows operating systems. A s Windows operating systems have improved, changed, and has become more secure, our v67 applications have become increasingly more difficult to support.  Issues related to the installation, performance, stability of Microvellum v6.7 software seem to be increasing as Windows advances. Now, with the release of  Windows 11 , we have decided it was time to end support for v67 entirely.  

By focusing development efforts around the latest Microvellum Software versions, our teams can concentrate on supporting Microvellum products built on and supported by the latest versions of Windows OS and AutoCAD. 

How Are Users of Microvellum v6.7 Impacted?

  1. If you have Microvellum v6.7 software already installed, it will continue to function.
  2. Activations for Microvellum v6.7 software will no longer be accessible as of December 1, 2021
  3. Support Contracts for clients that are only using Microvellum v6.7 applications will be no longer be renewed as of November 1, 2021.
  4. User support for Microvellum v6.7 software will end on December 1, 2021. This includes phone, email, support related to any areas of troubleshooting issues related to v6.7 installation, activation, and usage.
If you are currently running Microvellum v6.7 software, contact your Account Manager as soon as possible to discuss options for moving forward. If you’re not sure who your Account Manager is, you can find their contact details in the Help/About section of your Microvellum software application.
