Microvellum Foundation Library Release Notes | Build 25.0703

The following release notes apply to the Microvellum Foundation Library build 25.0703.

Additions

  1. Added a rebate (rabbet) to the bottom edge of top Nailers to create a notch that functions as a Hanging Cleat. This feature is globally controlled and can be enabled per cabinet type (Base, Tall, Upper). The depth and width of the rebate is globally controlled as well. Added five new Global variables: 
    1. “Top Nailer Rebate Width” 
    2. “Top Nailer Rebate Depth”
    3. “Rebate Upper Top Nailer”
    4. “Rebate Tall Top Nailer”
    5. “Rebate Base Top Nailer” 
  1. Added a new product, the Master Diagonal Vertical Designer Cabinet. 
  1. Added a new Miterfold Edge option to Adjustable and Fixed shelf interior components.  
  1. Added Drawing Icon for easier access to “2D Instructions" subassembly prompts. 
  1. Added five new global variables for additional control over CAD Dim Styles and Section Text Layers: 
    1. “Dimension_Style_Options” 
    2. “Plan_Dimension_Style” 
    3. “Elevation_Dimension_Style” 
    4. “Section_Dimension_Style”
    5. “Section_Text_Layer”
  1. If any of the above are set to "Template", the corresponding 2D drawing token will default to the dimension style specified in the drawing template (e.g., MV Annotative). 
  1. Text styles are currently setup throughout the 2D drawing tokens. Many of the subassemblies had hard values for the layer. We have gone through most of them and fixed this. 
  1. “Filler Text Alignment” and “Finished End Text Alignment” have been changed to a dropdown Options: “0|1|2”. This has been corrected as before we had it as an offset. 
  1. Added several new enhancements to the Wall room component:*
    1. Added the ability to have up to 4 windows and 4 doors per wall. 
    2. Added plumbing for waste pipes and tap outlets in the wall. 
    3. Added the option to have up to 4 electrical outlets in the wall. 
    4. Added elevation dimensions to the wall and elevation label. Users can either use the system wall variables or turn them on in the Wall product for better control. 
  1. Added Rakks Vanity Brackets to the ADA Sink Angled Frame product.  
  1. Added features to the Bracket products: 
    1. Added A & M Concealed Support Brackets to the Bracket subassembly. New materials added. 
    2. Bracket Array and Bracket Single products now both use the Bracket subassembly for easier maintenance. 
    3. Improved the structure to support both generic brackets and name branded brackets via a LookUpTable. 
  1. Added Electric Cooktop option to counter tops. There are two default Custom Cooktop options and 5 new branded models with DWGs. Plan views are also created. 
  1. Added additional leg lever, Hafele Axilo Adj Feet 35mm. 
  1. Added additional fastener, Hafele Zipbolt UT 35 Mini. 
  1. Added new Kessebohmer Magic Corner subassembly pullout. 
  1. Added a new global variable “Blind Corner Hinge Plate Distance From Edge”. This allows control of hinge plate locations from edge, specific for blind corner conditions. 
  1. Add the ability to count screw quantities for the Cabinet Connector feature. Added new hardware wizard variable “Cabinet Connector Screw Fitting”. Updated the global “Connecting Hole Diameter” to point to the appropriate LookUpTable. Note that previous diameter you have set may be changed after the update. 
  1. Added a “Slide Setback” prompt variable to the wood drawers. This will allow specific drawer system adjustment or specific user calibration. 
  1. Added Blum Tandem Slide Spacer hardware. 
  1. Added grain matching feature to all Scribe Filler products and subassemblies. 
  1. Added features to the Valance product: 
    1. “Cleat On/Off option”
    2. “Cleat Width variable”
    3. “Valance Top” vs “Valance Bottom”
  1. Added two new global variables to better accommodate hinge placements for small doors: 
    1. “Small Door Threshold” - Defines the maximum height at which a door is considered "small." 
    2. “Hinge Location Small Doors” - Specifies the reduced distance from the top and bottom edges for hinge placement on small doors.
    3. If a door’s height is less than the "Small Door Threshold", the hinge positions will be calculated using the Hinge Location Small Doors value instead of Hinge Location Bottom and Hinge Location Top. 
  1. Added new global variable “Dado Clearance Position Regular Dado" to provide better control over dado clearance gaps and which direction they go. The settings below determine how the clearance is oriented relative to the cabinet part receiving the dado: 
    1. Outside – Applies the clearance to the outer side of the panel (away from the cabinet's interior). 
    2. Inside – Applies the clearance to the inner side of the panel (facing the cabinet interior). 
    3. Both Sides – Applies an equal clearance on both sides of the dado, centering the cut. 
 

Modifications/Resolutions General

  1. Updated all door and drawer subassemblies by populating the Hide On Reports column for prompts that are unlikely to be used in reporting. This will reduce the amount of data populating the work order database and potentially decrease processing time. 
  1. Resolved an issue with Undermount Rangehood Cabinet. The product now utilizes the Undermount Strip Lighting Insert subassembly, which resolved an issue. 
  1. Resolved a regression issue with the Shelves with Shelf Std product. The “Product Type” prompt has been added. 
  1. Improved the formula logic that determines when a shelf exceeds the global max span distance, no matter the shelf location. Center fixed shelves will now follow the max span rule and change to a wide span material as required.  
  1. Added groove machining to the Grass ZBox drawer bottom. 
  1. Resolved an issue with the “Splay Edge” feature on drawer fronts. 
  1. Resolved several issues with the Cabinet Faces Only product: 
    1. Added missing “Product_Type” variable. 
    2. Corrected drawer front calculator issue. 
    3. “Use Aluminium Handle” prompt added. 
  1. Resolved the following issues with Five Piece Door subassembly:
    1. Flat center panel will now have the correct Y origin. 
    2. 2D elevations will now show the door outline when door construction is set to “Single Panel”. 
  1. Resolved an issue with Peka Base Pullouts. When changing from Right to Left hand, the drilling will now be correct. 
  1. Resolved an issue with the LeMans II Pullout related to regional and face frame differences. 
  1. Updated the remaining waste bins to include the 2D help text warning showing why it does not fit. 
  1. Updated the Kessebohmer Magic Corner II and graphics. The hardware will no longer have front drilling, as this is behind doors. 
  1. Added the Kessebohmer Magic Corner to the blind corner pullout list. 
  1. Resolved a naming issue for the Kessebohmer Conero 72H Comfort Tray Pull Out and Kessebohmer Conero 72H Internal Pull Out. The product names now align with both the drawing DWGs and the associated hardware names. 
  1. Resolved an issue with Wastebin Pullout prompts. The issue was caused by the Control type date using data from “G!Default_Double_Waste_Bin_Pullout”, when it should have been using “G!Waste_Bin_Pullout_Options”. 
  1. Resolved an issue with Upper cabinets. Selecting the Lift Up face option will now always work. 
  1. Resolved an issue with 2D Plan Dimension tokens involving Applied Ends and Fillers.  
  1. Resolved an issue with 2D Elevation tokens for cabinets with Applied End option.  
  1. Resolved regression issues with several subassemblies — including Any Angled Shelf Fixed, Angled End Shelf Fixed, Diagonal Angled Shelf Fixed, Angled Shelf Fixed, and Open Corner Shelf Fixed. These issues arose following updates made to the Shelf Fixed subassembly during the implementation of the magic wire options. At that time, prompt information was copied into these additional subassemblies, even though they do not support the magic wire feature. 
  1. Improved Institutional Hinges so they now respond to the “@” symbol (with distance value) in the hinge name. If the symbol is not there, the “Hinge Distance From Side” will be 0. 
  1. Simplified control for Double Division in 3-Door Cabinets. The "Double Mullion" prompt is now the primary control for enabling the double division feature in 3-door cabinets. When selected, it will automatically activate the double division option within the Horizontal Splitter subassembly. Previously, the feature was controlled through the "Double Division" prompt, which was also linked to the “Double Mullion” setting. This setup created confusion, as users had to check both prompts to confirm the feature was turned on. 
  1. The Component Folding Doors subassembly will now draw correctly in 2D Elevation. 
  1. Resolved an issue with the Access Panel subassembly. The "Access Panel Depth" was hidden instead of the "Access Panel Width" when using "Place Left Side" or "Place Right Side". Front to Back should use "Access Panel Depth", Left to Right should use "Access Panel Width", and Bottom to Top should use "Access Panel Height". 
  1. The ability to resize drawer parts with a rebate if they exceed 16mm has been added to other runners such as Blum MOVENTO.  
  1. Resolved an issue with the Titus Tekform Slimline drawer system. “Space Required Under Drawer Bottom” changed from 32mm to 26mm to match manufacturer specs. 
  1. Updated the Harn ALUNAN T and Harn Ritma Cube to include new and accurate bottom drilling as provided by Harn. 
  1. Resolved an issue with Wood Drawer Box subassemblies when using the “Plant On Bottom” feature. Slide holes will now remain in the correct location. 
  1. Improved drawer rear cutout feature so drill machining remains on face 5 only. 
  1. The “Screw Construction” for the Horizontal Splitter subassembly has been updated to accommodate the small part threshold. 
  1. Changes and improvements made to the Internal Drawers With Faces component insert: 
    1. The drawer material is now linked to the OpenID. This ensures that when the insert is placed into a standard cabinet, the drawer fronts will automatically default to the carcass material. 
    2. A new “Filler Material” prompt has been added, providing control over the side filler material directly through prompts. 
    3. In the Finish Faces subassembly, a “Finish_Faces_Layer” prompt has been introduced. This allows the layer assignment to be controlled via prompt values. 
  1. Improvements made to the Drawer Box List Report.  
  1. Resolved an issue with cut size values appearing incorrectly on Part Labels when the processing station Premill Type is set to “Edgeband Thickness”. All labels have been updated to use the Adjusted Cut Part sizes so the size will account for the correct premill value. 
  1. Altered the way Mitered Side Front Edges behave on Corner Diagonal cabinets. When this option is selected, the left and right cabinet depths now reflect the outside edge of the mitered sides. Side widths now include the angular projection. 
  1. Resolved issue in Corner Cabinets involving the rear corner notch. Parts will now avoid face 6 machining for the drill holes. 
  1. Added missing graphics for Olympus 700S locks. 
  1. The prompts in the Project Wizard for “Base_Door”, “Tall_Door”, and “Upper_Door” have been updated to represent a buyout material when ticking the “Buyout_Door” prompt. 
  1. Updated the Chamfer Cabinet Modifications for the left, right, and both sides to work correctly when checking the finished Chamfer Left or Right. This will now work with different thickness sides vs the back. 
  1. The “Edit Part Materials” prompt has been fully removed from all products and subassemblies. This was a legacy prompt that no longer served a meaningful function in the current product structure. Historically, the prompt was used primarily to disable the material prompts, and the thickness prompts would look directly at the material pointer when it was unticked. However, it was inconsistently applied—none of the face frame products utilized it, and most thickness prompts did not reference it. As part of this cleanup: 
    1. All control types and material pointer references still relying on “Edit_Part_Materials” have been updated. 
    2. Thickness prompt formulas were simplified as part of this removal, improving overall clarity and reducing unnecessary dependencies. 
  1. Resolved machining issues and added improvements to the ADA Sink Angled Frame product.  
    1. Angled Route Polylines: Polylines have been updated so that when the corner radius is set to 0, the associated corner radius vector is now fully removed from the route. Previously, a 0-length vector remained, which caused issues during machining or toolpath generation. 
    2. Independent Radius Control: All corner radii in the polyline can now be toggled independently. 
    3. Toe Kick Radius Option: A new prompt has been added to control the internal radius where the bottom of the panel meets the top of the toe kick. This was previously tied to the general corner radius, but now you can adjust it separately for more precise detailing. 

Additions/Resolutions Face Frame Cabinetry

  1. Added new products Master Corner Vertical Designer Cabinet FF & Master Diagonal Vertical Designer Cabinet FF. Along with the new designer products, several subassemblies have been created:
    1. Component Corner Doors FF 
    2. Component Diagonal Doors FF 
  1. Face Frame Shelves - These shelves have been created as bay splitters behind the face frame mid rails. They act the same as a vertical splitter, with a small difference; they are designed to be placed behind mid rails. With the ability to change the fixed shelf condition behind the rails, with options: No Shelf|Centered|Top|Bottom|Double 
    1. Corner Shelf Fixed FF 
    2. Diagonal Angled Shelf Fixed FF 
  1. You can control the position of the shelf using these conditions: 
    1. No Shelf: Disables the face frame shelf entirely—no splitter is placed. 
    2. Centered: Places the shelf centered directly behind the mid rail.
    3. Top: Aligns the shelf to the top edge of the mid rail. 
    4. Bottom: Aligns the shelf to the bottom edge of the mid rail. 
    5. Double: Places two shelves—one at the top edge and one at the bottom edge of the mid rail. 
  1. The Diagonal Angled Face Frame, Corner Face Frame, and Diagonal Face Frame have been updated to accommodate the mid rails in the new designer cabinets. 
  1. Improvements made to Component Drawers FF: 
    1. Draw Intermediate Rails: Enables drawing and machining of each intermediate rail. If unchecked, rails are omitted, but drawer spacing remains intact. 
    2. Intermediate Rail Extension Left / Right: Extends the intermediate rail into adjacent bays, allowing alignment or bridging across openings. 
  1. Resolved a Face Frame overlay issue in the Master Split Height Designer Cabinet FF. The Component Drawers subassembly in the first bay had an issue with the “Face_Frame_Right_Overlay”. When using “Face_Frame_Division_Overlay_Control_Stile_1” ="Right", it was not deducting the “Thickness_Division”. All the other bays are correct. 
  1. Resolved an issue with Master Horizontal Designer Cabinet FF. Interior components will now properly follow the correct Face Frame Division Overlay Control for each bay. 
  1. Resolved an issue with the 2D Draw tokens in the Base Corner Cabinet FF. The depth dimension was accounting the “Applied_End_Thickness_Right” value. When used in the face frame products, should just be the depth. 
  1. Updated all FF Drawer Bank Cabinets to have the same drawer calculator order as the frameless cabinets. (Top, Bottom, 2nd, etc.). This order will ensure accuracy should the user change the number of drawers after calculating. 

*Not included in Library Update, only in the full install.
