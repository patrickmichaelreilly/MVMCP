Toolbox BSB: Supported File Extensions for Import/Export

Toolbox BSB, using BricsCAD Pro as its basis, offers full support for the same wide range of file types for import and export. Certain file formats not listed in this article can be imported/exported by BricsCAD, but are not, at the time of writing, available for import/export in Toolbox BSB. These file types may become available in future versions of the software. 

CAD File Formats

CAD formats are file types that are either native to or available for usage in CAD software. Toolbox BSB supports the following file types for import or export. File types are marked according to whether they can only be imported or exported, or both. File types created specifically for use with BricsCAD (such as settings files or are designated as native files in the Import/Export column. Some file types capable of being imported (such as SketchUp files) may be imported in a state that requires an additional step to be usable in BSB (SketchUp files are imported as meshes, and require one to use the SOLIDIFY command, as noted below).

Vector Formats

Vector files are images coded with geometric formulas to enable them to be scaled with minimal quality loss. BricsCAD allows for the import and export of multiple different vector file formats. Specific formats have additional functionality tied to their import, as noted in the table below.

Point Cloud Formats

Point Cloud files contain data points collected for the purpose of analyzing geographical regions, terrain, buildings, or space. Toolbox BSB, being built using BricsCAD Pro, allows for the import and export of multiple different Point Cloud files. Generally, all the file types listed below can be imported into BSB using the POINTCLOUDREFERENCE command, then selecting to import a file.
