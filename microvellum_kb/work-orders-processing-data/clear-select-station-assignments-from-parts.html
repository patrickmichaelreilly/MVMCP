Clear Select Station Assignments from Parts

A new feature in the Processing Center interface allows for the removal of specific Processing Station part assignments as opposed to removing all of the Processing Station assignments for the selected part(s).
  1. From the Processing Center, select the part in the interface from which you wish to remove the Processing Station assignment.
  2. Select ‘Clear Select Station Assignments from Parts.’

  3. Select the Processing Station you wish to remove from the Processing Station Selection interface.

    4. Multiple selections are permissible from the Processing Station Selection interface.

To remove all of the Processing Station Assignments from a selected part, select ‘Clear All Assignments from Selected Parts.’
