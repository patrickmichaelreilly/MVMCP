Employee Permissions & Software Security (Reference)

This article contains a reference for each employee permission available in the sections Module Access and Spec Groups to help you decide what permissions to grant to which employees.
Unless noted below, these permissions apply to both Production and Toolbox.

General tab


Fig. 1 - Use the General tab of the Employee Permissions box to establish the types of operations each User is permitted

Module Access

Access Estimating
Grants the user access to the Estimating module to create and edit material costs (only Microvellum Production Modules [Production]).
Access Scheduling
Grants the user access to the Scheduling module to create and edit time entries and schedules (only Microvellum Production Modules [Production]).
Access Production
Grants the user access to the Productions module to create and edit Work Orders.
Access Purchasing
Grants the user access to the Purchasing module to create and edit Purchase Orders (only Microvellum Production Modules [Production]).
Access ReportDesigner
Grants the user access to the Report Designer.

Options

Access General Settings
Grants the user access to the settings on the options page General tab.
Access Machining Settings
Grants the user access to the setting on the options page Machining tab.
Access Data Settings
Grants the user access to the settings on the options page Data Access tab.
Access Library Designer
Grants the user access to edit Library in Library Designer.
Access Database Explorer
Grants the user access to Database Explorer to import and export spreadsheets.
Access Database Utilities
Grants the user access to Database Utilities to import and export Libraries, Projects, Materials, Reports, Tool Files, Processing Stations, Work Orders, and Other Utilities.

Data Setup

Access Transfer Tables
Grants the user access to transfer tables functionality to transfer data from the SQL Database to a Local database and vice versa.
Access Setup
Grants the user access to Toolbox Setup and Options.
Import Data
Grants the user access to the Import commands found under Toolbox Setup.
Access Tool File Management
Grants the user access to modify tool files.
Access Processing Station Management
Grants the user access to modify processing stations.

Manage

Access Time Clock Management
Grants the user access to edit Time Clock entries (only Microvellum Production Modules [Production]).
Access Company Management
Grants the user access to set up Employees and their information.

Save

Save Products to Library
Grants the user access to save custom products to the Library.
Save Subassemblies to Library
Grants the user access to save custom subassemblies to the Library.

Notifications

Allow Edit Incoming
Grants the user access to edit incoming Microvellum Server Notifications.
Allow Edit Outgoing
Grants the user access to edit outgoing Microvellum Server Notifications.

Update Access

Allow User to Receive Updates
Grants the user access to update from the Login Screen when checking for updates.

Event Logs

Allow Archive
Grants the user access to archive logs from the event viewer.
Allow Delete
Grants the user access to delete logs from the event viewer.
