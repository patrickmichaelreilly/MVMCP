Permissions Required by Toolbox

What are the User permissions required by Toolbox?

Toolbox software (the User) needs full permission to: 

Registry Entries:

  • HKEY_CURRENT_USER\SOFTWARE\Microvellum
  • HKEY_LOCAL_MACHINE\SOFTWARE\Microvellum

Folders:

  • C:\ProgramData\Autodesk
  • C:\ProgramData\ Microvellum
  • C:\Program Files\ Microvellum (or wherever Toolbox was installed)
  • C:\Program Files\Common Files\Autodesk Shared
  • C:\Users\{user}\AppData\Roaming\ Microvellum
  • C:\Users\{user}\AppData\Local\ Microvellum
  • C:\Users\Public\Documents\ Microvellum
  • C:\Users\{user}\AppData\Local\Temp 

Network Share:

  • Path To Microvellum Data

SQL Database:

  • Full permissions needed for SQL Users/Groups (dbo)
  • Groups need to be set to use dbo Schema

Depending on how your IT has your Group Policy set up, some of these may need to be double-checked and corrected after install as well as updated in Group Policy.
