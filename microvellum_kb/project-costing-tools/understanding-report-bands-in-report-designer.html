Understanding Report Bands in Report Designer

The articles in this section of the Knowledge Base involve working with Report Bands for display in your reports. To access the Report Designer, select Toolbox Setup > Design Reports > Design (Report Type) Report. Select New Report to design a report from scratch or New Report from Existing. Select Designer to open Report Designer.

From this screen, add a Report Band component to your report, or edit an existing report band component.


Fig. 01 – Report Bands

The Report Designer supports two report bands: the Reports Summary and the Report Title bands. The Report Summary band is displayed at the end of a printed report, and the Report Title band is displayed at the beginning of a printed report. The Report Summary and Report Title bands can be output more than one time and can be used on each page.

There is no limit to the number of Report Summary and Report Title bands on a page.

Report Title Band

The report header can be displayed using the Report Title band. The report header is only displayed once at the beginning of the report. The Report Title band comes after the Page Header band, and before the Header band.


Fig. 02 – Example with Report Title Band

Figure 02 shows how bands can be placed on a page. From the top-down, there is a Page Header , Report Title , Header , and Data band.

Report Summary Band

The Report Summary band is displayed on each page as many times as there are pages. It is used to output a report summary.


Fig. 03 – Example with Report Summary Band

Figure 03 shows how bands can be placed on a page. From top-down, there is a Page Header , Report Title , Header , Data , Footer , and Report Summary band.
