Previously, there were two ways to determine the format mask:
  1. Local Settings ‚Äì Using Local Settings, the text is formatted with the settings currently applied to the operating system.
  2. The format mask is manually applied to each parameter.

There are disadvantages to both of these methods of determining the mask, but there is a third option. The local settings can be used to change any parameter format. Set the value of the parameter after you‚Äôve set the flag next to it.


Fig. 01 ‚Äì Text Formatting > Percentage Tab

  1. Use Group Separator ‚Äì When the group separator is used, then currency values are separated into number positions. 
  2. Use Local Settings ¬≠‚Äì When using the Local settings, currency values are formatted according to the current OS installations. 
  3. Negative in Red ‚Äì When using the Negative in Red parameter, negative numbers are printed in red. 
  4. Decimal Digits ‚Äì Number of decimal digits, which are used to format currency values. 
  5. Decimal Separator ‚Äì Select the delimiter to be used as a decimal separator. 
  6. Group Separator ‚Äì Select the delimiter to be used as a group separator. 
  7. Group Size ‚Äì The number of digits in each group in currency values formatting. 
  8. Positive Pattern ‚Äì This pattern is used to format positive numbers. 
  9. Negative Pattern ‚Äì This pattern is used to format negative numbers. 
  10. Percentage Symbol ‚Äì The symbol used as a percent sign.
