How to Modify Subassemblies through the Prompts Interface

The majority of the library products contain subassemblies. Subassemblies are simply products within a product. They contain their own set of prompts that are accessible by selecting Subassemblies from the menu located in the top left corner of the prompts (or by right-clicking in the blank area of the workspace and selecting Subassemblies). 

 
  1. Subassembly Library – Allows you to insert subassemblies to a product.
  2. Product Subassembly List – Displays the active subassemblies within a product and allows you to edit the parameters as needs arise.
  3. Formula Editor – Allows you to view and edit the values/formulas for a subassembly’s parameters.
  4. Quick Access Menu – The menu is accessed by selecting a subassembly and then right-clicking. The menu will display the options available for the selected subassembly. Clicking the Show Subassembly Prompts icon will allow you to make additional changes to products that aren’t available within the primary prompts interface.
  5. Color Legend for Subassembly Save Levels - The color key at the bottom of the interface describes color code for the different subassembly save levels.
The Subassembly Prompts interface is similar to the Product Prompts, except for some of the fields that have been removed and the addition of save options (located in the bottom right corner).

There are three save options available through the Subassembly Prompts:
  1. Save this Subassembly Only – Allows the subassembly to be unique from any other subassembly of the same name.
  2. Save Subassembly to Product – Allows the subassembly and any other subassemblies of the same name within the product to have the same changes applied.
  3. Save Subassembly to Project – Allows the subassembly to be saved across the entire project. Any subassembly, regardless of what product it resides in, will inherit the changes applied.
