How to Draw Walls in Toolbox

Drawing Walls

To begin drawing walls, click the Draw tab in Toolbox and then click the Draw Room Components tab. By default, the Walls category will be displayed; if the Walls category is not shown, click on the drop-down menu.  

By selecting the Wall product and right-clicking (or double-clicking), you can access the quick menu. 


  1. Pick Points to Draw Wall – Allows you to draw walls by picking points in model space.
  2. Select Polyline Path for Wall – Allows you to draw walls by selecting a 2D polyline to follow.
  3. Properties – Allows you to access the prompts interface for a selected wall.
  4. Erase* – Allows you to erase a wall.
  5. Move* – Allows you to move a wall.
  6. Rotate* – Allows you to enter a value to rotate the wall along its initial base point.
  7. Copy* - Allows you to copy a wall.
  8. Adjust Wall Length – Allows you to adjust the length of a selected wall. The products associated to that wall can have the widths automatically recalculated.
  9. Associate Products to Wall – Allows you to associate products to a wall that were not previously attached to a wall or have become disassociated.
  10. Redraw 2D Elevation – Allows you to redraw a 2D elevation of a selected wall.

Using AutoCAD commands – ERASE, MOVE, ROTATE, and COPY – will only change the way the drawing appears. AutoCAD commands, when working with 3D Microvellum products, will not update the Microvellum database. Using AutoCAD commands to modify 3D Microvellum products will cause adverse effects when products are redrawn. When working with 3D Microvellum products, use the Microvellum commands to ensure the drawing and the database are updated correctly.

To draw a wall using Pick Points to Draw, simply click points in the AutoCAD model space. After selecting the first point, each point after will create a new wall segment. Points should be selected in a clockwise direction. 

To draw a wall using Select Polyline Path for Wall, you will need to draw a polyline in the model space first. Once a polyline has been drawn, you can select it, and the walls will be drawn along the polyline. Polyline points should also be selected in a clockwise direction.
