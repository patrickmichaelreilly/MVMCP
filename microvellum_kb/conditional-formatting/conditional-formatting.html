Conditional Formatting

A user can change the appearance of components by setting specific conditions. Conditional Formatting is set up using the Conditions window, found in the Home tab of the ribbon.


Fig. 1 – Home > Conditions


Fig. 2 – Conditions Window

  1. Add Condition – Adds new conditional formatting to component conditions.
  2. Remove Condition – Removes conditional formatting from component conditions.
  3. These arrows allow the user to move the conditional formatting up or down in the list. The level in the list is directly related to the priority of processing. The higher an item in the list, the higher the priority of processing.
  4. Break if True – Conditions will be processed sequentially from top to bottom. To break out of processing all conditions, you can use this setting to cease processing when the component evaluates as true for the condition. This will create a loop until the component evaluates as true.
  5. Add Level – Used to add a single level of the condition parameter.
  6. Parameters of a Condition – Use this panel to specify the specific parameters of a condition.
  7. Format Settings – Use this panel to create a specific appearance for a component.
