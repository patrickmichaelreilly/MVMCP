Face Frame Expansion Overview

Face Frame Expansion Overview

The Microvellum Face Frame Expansion for the Foundation Library contains a broad collection of various Face Frame cabinet products intended to be used as a starting point for manufactures or create their own catalog. The expansion contains 113 unique products and 25 unique subassemblies that use existing global variables for customizing desired details. Each product also contains dozens or more variables unique to each product for further customization.

For installation and purchasing, please contact your account manager. 

Master Products

Master Cabinet FF




Description: Single modular face frame cabinet product. This is a “Master” product capable of being saved back to Library numerous times in unlimited number of different combinations.  This product is included with the Foundation Library Face Frame Expansion.

Main Options: 
  1. Parametric Sizing Width/Height/Depth/Toe Kick Height
  2. Specify Left/Right End Conditions (None/Applied End/Face Frame)
  3. Left/Right Filler variables
  4. Specify Rails/Full Top/Overlay Top
  5. Specify No Bottom/Full Bottom/Overlay Bottom
  6. Specify Finished Left/Right/Back/Top/Bottom
  7. Countertop option with variables tab (generally for drawing representation)
  8. Access to All Construction Methods, Material Options, Advanced Options, Flatshot Options, 2D Drawing Options, Item Number Options, and Rendering Options

Internal Options:

  1. Access to all Component Insert Options
  2. Continuous Shelf Holes option
  3. Add Center Shelf option
Face Options: 
  1. Access to all Face Options (No Door/Left Swing/Right Swing/Two Doors/Liftup Door/Drop Down Door/Drawers)
  2. Access to all Foundation Library Door Types & Styles
  3. Overlay/Full Inset Door options with overlay/gap variables
  4. Top Drawer & Double Top Drawer option
  5. Add Top Facia option
Face Frame Options: 
  1. Face Frame Options (Standard/Separate Bead/Haunched/Single Panel)
  2. Left/Right Stile Scribe variable for wall or corner conditions
  3. Frame Left/Right Side Overlay Control (Inside/Outside)
  4. Frame Top/Bottom Overlay Control (Inside/Outside)
  5. Frame Left/Right/Top/Bottom Overlay variables
  6. Frame Stile & Rail width variables
  7. Front Left/Right Corner Choices (Standard Left/Corner Notched/Chamfered/Radius)
  8. Frame Stile Left/Right removal options (to use adjacent cabinet stile)
  9. Add Intermediate Stile option (when two doors are present)

Hardware Options: 
  1. Ability to change the default Handle Type
  2. Handle Distance from Side/Top/Bottom variables
  3. Horizontal Handle option
  4. Add Lock option
  5. Add Wall Hanger option
Additional Info:
As with any “Master” cabinet product in the Frameless Cabinetry category, each can become a Base, Base Open, Tall, Tall Open, Upper, or Upper Open by the Product Type combo box inside the Advanced Construction Options tab. In the same tab, we can also control the desired construction method for each of the various cabinet parts.

Inside the Material Options tab, we can individually specify a different material for each of the cabinet parts if they differ from what is set in the specification group. We can also control the grain direction for doors, sides, and cabinet backs.
