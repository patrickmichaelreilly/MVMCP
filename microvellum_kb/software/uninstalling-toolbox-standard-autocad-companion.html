How to Uninstall Toolbox Standard (AutoCAD Companion)

These procedures explain the removal of Toolbox Standard (AutoCAD Companion) and supporting files while using Windows 10.

 

Uninstalling Toolbox Standard from “Apps and Features”

  1. Select the “Windows start icon”.
  2. Select “Settings”.
  3. Select “Apps”.
  4. In “Apps and Features” scroll down until you find “Microvellum Toolbox Standard x64” and select it.
  5. Select “Uninstall” through all the prompts.
  6. You will be notified with the message “You have successfully uninstalled this product” when Toolbox AC has been uninstalled.

Removing Microvellum Data from Your Registry

WARNING – Microvellum is not responsible for modifications made to the registry. Improper edits may render an operating system unusable. For more information visit Microsoft support.
  1. Log on the workstation as a member of the local Administrators group.
  2. Launch the Run dialog by typing [Windows key] + R.
  3. In the run dialog, enter REGEDIT in the entry box and select ok.
  4. In the Registry Editor, locate and delete the following subkeys. WARNING – This will delete all remaining Microvellum data.
  5. HKEY_CURRENT_USER\Software\Microvellum
  6. HKEY_LOCAL_MACHINE\Software\Microvellum
  7. Close the Registry Editor and restart your computer.
