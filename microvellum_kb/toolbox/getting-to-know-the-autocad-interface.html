Getting to Know the AutoCAD Interface

AutoCAD is the drawing platform for Microvellum. The AutoCAD interface can be completely customizable to suit your needs. The descriptions and images shown below are just an example of how AutoCAD can be configured. It is recommended to research best practices and options for customizing the appearance and accessibility of AutoCAD and its commands and tools.



  1. Ribbon - The Ribbon can be set to show tabs, panel titles, or panel buttons (as displayed). The ribbon can be closed, and toolbars added.
  2. Drawing/Model Space - The drawing/model space is where most of the drawing operations occur.
  3. Command Line - Allows you to type in commands and also displays the next step in other commands. It is beneficial to look to the command line often.
  4. View Cube - Allows you to change the view of the drawing space.
  5. Model and Layout Tabs - The Model tab will display the drawing space, as shown. Layout tabs are individual sheets (paper space) and can be set up to view specific areas of the drawing in model space.
  6. Status Bar - Contains settings that affect drawing operations and properties.
  7. Toolbox - The Toolbox is shown docked to the left in the image above. The Toolbox can also be docked to the right if desired.
