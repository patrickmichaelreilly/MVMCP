Understanding the Toolbox Login Screen

The Login Screen opens when Toolbox is launched and provides important software details and tools, as well as optional secure login functionality.  This article contains an overview of the contents of the Login Screen.


  1. Microvellum Product – Indicates which Microvellum product is installed.
  2. Product Version – Indicates the currently installed version (build number) of Microvellum Toolbox. Learn more about Microvellum's Software Versioning Schema.
  3. Session Management – Opens the session management interface (FLA type licenses only), allowing you to check out a session to work offline for a duration of up to 7 days.
  4. Refresh License – Refreshes your license, re-establishing the license's credentials and reapplying all existing associations to that license.
  5. Deactivate License – Deactivates the current license on the device you're using. A pop-up will prompt you to confirm this decision before the current license is deactivated. 
  6. Configuration Selection – Allows you to select which configuration to use when launching Toolbox.
  7. Manage Configurations – Opens an interface that will allow you to manage your configurations and preferences.  Check out our Introduction to the Microvellum Configuration Manager for more information.
  8. User Name/Password – When security permissions have been set up, allows you to enter your user name and password to login into the software.
  9. Microvellum Knowledge Network – Find out more about your Microvellum software by searching our knowledge base, submitting a support ticket, or connecting with the community by clicking the Microvellum Knowledge Network link.
