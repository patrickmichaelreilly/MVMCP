Edgeband Premill Settings and Part Size Adjustment

In the processing station set up, there are three options for Edgeband Premill.

Use Legacy Premill

This setting (Figure 1) will cause the program to remove from the part edge the thickness of the edgeband material applied to that edge while adding back the amount specified by the "Edgebander PreMill Dimension" found in the toolfile (Figure 2).


Fig. 1 – Use Legacy Premill



Fig. 2 – Edgebander PreMill Dimension

Edgeband Thickness

This setting will cause the program to increase the cut size of the edgebanded edge of the part by the thickness of the edgeband material applied to that edge – it doesn't look at the premill value set in the toolfile.


Fig. 3 – Edgeband Thickness

Custom Edgeband Amount

This setting will cause the program to increase the cut size of edgebanded edge of the part by the "Part Size Adjustment" amount specified for the edgeband material in the Material File (Figure 5). 

This setting also doesn't look at the premill value set in the toolfile.


Fig. 4 – Custom Edgeband Amount



Fig. 5 – Part Size Adjustment


This setting allows more control than a single premill setting, as is used in the "Legacy" setting.

With this setting, the program will reduce the part size by the edgeband thickness amount but will adjust the size based on the "Part Size Adjustment" amount. This allows control over the premill specific to the edgeband material.

For example, if your edgebander removes the edgeband thickness amount before applying the banding, this premill amount will need to adjust based on the thickness of the edge material. With a single "Edgebander PreMill Dimension" setting, you don't have this flexibility.

Using the above example, the edges that contain the PL1 PVC banding will be reduced by .02. However, .04 will be added back to the edgebanded edge containing this material, adding net .02 back to the part.

In the example below (Figure 6), the edge containing WD1 Veneer will be reduced by .03. However, .06 will be added back to the edgebanded edge containing this material, adding net .03 back to the part.


Fig. 6 – Part Size Adjustment Example


For "Part Size Adjustment", enter the premill amount that you want to add to the edgebanded side of parts. This gives you more control over the premill amount specific to the material, allowing you to adjust the premill based on the edge thickness, instead of a static premill.
