Edit a Material Size

This article explores how to change the sheet sizes for a given Material in the Material File of Toolbox.
To change the sheet size for a desired material: 

1. Open the Library or Project Specification Group interface from the Toolbox Setup dropdown menu.

2. Select the Material File icon in the Navigation Menu.

3. Navigate to the Material you wish to change in your Library and right-click on the Material. Select Edit Selected Material. 

4. In the Edit Material interface, double-click or select "Edit Sheet Size" to open a dialog box where you can change the panel width or length, width or length trim, optimization priority, and quantity availability.



5. You also have the ability to add an additional sheet size if you wish to have different Optimization Priorities for multiple sheet sizes.
