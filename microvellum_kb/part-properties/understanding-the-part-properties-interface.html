Understanding the Part Properties Interface

Understanding the Part Properties Interface

The Part Properties can be accessed through the Toolbox Modify tab, and then the Modify Products tab. It is also preloaded in the customizable tool strip.

To view the interface, click on the Part Properties icon, and then select a product.


  1. Subassembly List – This section displays all the subassemblies within the product. Subassemblies with a green, yellow, or red folder are active (quantity greater than 0), and subassemblies with are grey folder have a quantity of 0. Selecting a subassembly and then right-clicking will allow you to “Show Properties,” which will load the subassembly and its parts, or “Revert a Subassembly,” which will enable you to restore a subassembly to the project level or from the library.
  2. Part Tabs – Cycling through these tabs will display the properties for the part selected in the Part List.
  3. Work Area – This section displays all the properties available within the selected tab for the selected part.
  4. Part List – This section displays all the parts within a product or subassembly.
If no parts are displayed, like the image above, this does not mean the product does not contain any parts. A subassembly will need to be loaded to populate the part list.
  1. Additional Part Tabs – Cycling through these tabs will allow you to access hardware, subassemblies, and prompts for the selected part.
  2. Formula Editor – This section allows you to add/edit a value/formula for a selected field.
