Applying a Sort to a Data Band in Report Designer

The articles in this section of the Knowledge Base involve working with Lists for display in your reports. To learn more about Lists and how to access the Report Designer, see Creating Lists in Report Designer. This article provides information for applying a Sort to a Data band.

Sometimes data, which is used for report rendering, is sorted in an order that does not meet your needs. If this is the case, data can be sorted using the abilities of the Report Designer. Each Data band can have Sorting applied to it separately. To set Sorting, open the Data band editor by double-clicking the Data band. It is also possible to call the Data band editor using the Sort property of the Data band (Figure 01).

Each Sorting can consist of several levels. For example, Figure 02 demonstrates two levels in the Sorting. This report is first sorted by the “Name” column, then by the “Width” column. There is no limit to the number of sorting levels. One level can be set to ascending and the next to descending.
