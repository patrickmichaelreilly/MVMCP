Creating Multiple Lists in Report Designer

The articles in this section of the Knowledge Base involve working with Lists for display in your reports. To learn more about Lists and how to access the Report Designer, see Creating Lists in Report Designer. This article provides information for adding multiple lists to one report.

Sometimes it is necessary to print multiple lists in a report. The Report Designer has no restrictions on the amount of Data Bands that can be included. The tutorial below stands as an example of this:

  1. Place two Data Bands on a page, and specify them with different Data Sources. In Figure 01, I’ve added Text Components and pointed them towards Data Columns.

    2. Fig. 01 – Two Data Bands in the Report Template

  2. Create a header and a footer for the Data Bands. When adding multiple lists, you will need to add multiple headers and footers. The Header Band should be placed directly over the Data Band it corresponds with, and the Footer Band should be placed under it.

    3. Fig. 02 – Multiple Lists – Note Header and Footer Band Placement

  3. Render the Report. The first Data Band renders the first list (Red). The second Data Band renders the second list (Blue).

    4. Fig. 03 – Multiple Lists Report Example
