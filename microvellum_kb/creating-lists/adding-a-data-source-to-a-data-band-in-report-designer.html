Adding a Data Source to a Data Band in Report Designer

The articles in this section of the Knowledge Base involve working with Lists for display in your reports. To learn more about Lists and how to access the Report Designer, see Creating Lists in Report Designer. This article provides an overview of adding a Data Source to a Data band.

To output lists using the Data band, it is first necessary to specify a data source. It is important because the Report Designer should know how many times the Data band must be printed. There are several ways to associate a data source to a data band. First, it is possible to use the Data band editor. To access the Data band editor, double-click the Data band. It is also possible to access the editor from the context menu (right-click and select Design). See Figure 01 for an example of this menu.


Fig. 01 – Context Menu

Also, the editor can be called using the Data Source property of the Data band.


Fig. 02 – Data Source Property

The Data Editor is where the data source is selected. All data sources are grouped into categories. Each category is one data connection with data in the Dictionary of Data. Figure 03 shows data in the Data editor.


Fig. 03 – Data Editor

  1. The Data Source is selected from this tab.
  2. Select this node if there is no reason to specify a data source.
  3. The “Demo” category of data.

The quick button can be used on the Data band to select the data source from the menu. The basic elements of the menu are represented in Figure 04.


Fig. 04 – Quick Button Access

  1. This is the quick access button that will drop-down the list of tables in the Microvellum Database.
  2. Use the Not Assigned selection to reset data source selection.
  3. The Products data source is selected.
