Configuration Editor: Region Options

This article provides an overview of the options and settings available on the Region tab of the Configuration Editor (Options) Interface.  For a complete list of available tabs and options visit Overview: Configuration Editor (Options) Interface.

Set region-specific properties such as the Interface Language, Spreadsheet Culture, and Geographic Region on this tab.


Interface Language

Set the language of the interface.


Spreadsheet Culture

Set the culture of the spreadsheet interface.


Geographic Region

Set the Geographic region where the application is used.


Translation Assistance

Note and submit custom translations not handled by the program interface language.
