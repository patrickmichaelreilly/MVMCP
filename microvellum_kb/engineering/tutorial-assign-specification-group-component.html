Tutorial: Assign Specification Group Component

This article demonstrates assigning a component to a specification group. 

  1. With Toolbox open, click "Toolbox Setup > Project Specification Groups."
  2. Select the spec group component, to replace, in the left spec group window. 
  3. Select the new component in the right component window. 
  4. Click the "Assign Component..." button in the center between the two windows (See the workflow below). 

  5. Take note that the unused spec group components will display as orange text (all components using the same source file will display as blue text). 

The use of the Assign button is similar to that used inside the material file for assigning pointers. 

A short video that demonstrates these steps is attached to this article.
