Foundation Library Catalog: Appliances

This document lists the full range of appliance products within the Microvellum Foundation Library. Appliances are non-cabinet products such as refrigerators, dryers, microwaves, intended to be placed into Microvellum project rooms for visualization and decorative purposes, with none of the products listed in this document producing any manufacturing cut parts or machining when processed in an order. Appliances, in the Foundation Library, can have their model and specifications altered by selecting from one of the different brands available. 

This document will reference “Standard Prompts”, referring to the set of prompts that are generally present in all products in the Microvellum Foundation Library catalog by default. These prompts will not be listed among the prompts of products, as they are considered default. Standard Prompts are the following:
  1. Width
  2. Height
  3. Depth
  4. Toe Kick Height
  5. Left/Right Filler Width
  6. Product Name
  7. Product Version
  8. Left/Right Finished End
  9. Left/Right Applied End
  10. Show Construction Options
  11. Show Material Options
  12. Show Advanced Options
  13. Show Flatshot Options
  14. Show 2D Drawing Options
  15. Show Item Number Options
  16. Show Rendering Options
Many of the appliances within this document have different options for the appliance's brand, including the option to create a custom brand. In the event one wishes to use a custom brand, the brand is capable of being added to the dropdown menu and customized via the product's design data. The LookUpTableAppliance table controls brand driven appliance options. If selecting from an existing brand, this table will set the appliance to use each brand's manufacturer specifications, setting dimensions, descriptions, and region availability. If one is creating a custom brand, it is recommended to copy the same format as the existing brands within the LookUpTable. 

Wall Oven

 
Product Category: Appliances

The Wall Oven Product Origin is Floor Height by default. The Product can be placed at any point in the room space. (i.e. at countertop height, on a shelf, etc.)

Product Description
The Wall Oven product is designed for user creativity to achieve a generic Wall Oven size and style while also offering the option to load a preset Wall Oven that is ready for instant selection and placement when needed. The Foundation Library contains a variety of Wall Oven options from two distinct brands: 
  1. Smeg
  2. Gaggenau
Basic Prompts/Options:
  1. Brand – Selection prompt for the user to choose between a “Custom” Product or a Brand selection.
  2. Appliance – This Dropdown prompt displays the list of options set in the LookUpTable. 

Unique Product Features/Prompts
The “Custom” product option is a user designed Wall Oven that is constructed with parametric DRAWBLOCK parts that do not produce manufacturing cut parts or machining. Each of these parts are controlled by the prompts available when the brand dropdown is set to “Custom.” The following prompts are available for customization when the appliance brand is set to "Custom":
