Overview: Configuration Editor (Options) Interface

The Configuration Editor (Options) Interface for Microvellum Toolbox is accessible from the Toolbox Setup menu. Using the options contained within, you'll configure general system settings, set up the optimization software you use, set up tool files, and access many tools related to populating data to and extracting data from the database.

The options dialog contains ten (10) tabs, and each tab contains multiple sections. Follow the links below to learn more about the options available on each tab.


Configuration Editor Tabs

  1. General
  2. AutoCAD
  3. Processing
  4. Machining
  5. Toolfiles
  6. Processing Stations
  7. Solid Analyzation
  8. Region
  9. Data Access
  10. Utilities
