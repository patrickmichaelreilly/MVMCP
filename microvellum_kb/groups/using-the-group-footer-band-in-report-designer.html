Using the Group Footer Band in Report Designer

The articles in this section of the Knowledge Base involve working with Groups for display in reports. To learn more about Groups and how to access the Report Designer, see Understanding Groups in the Report Designer. This article regards working with Group Footers in Report Designer.

The Group Footer band is used to display aggregate functions that are specifically applied to the data within each group, as opposed to a standard Footer band, which is applied to the data in the entire report.

For example, the following expression applied to a standard Footer band would count all of the Data Rows in the report. The same expression applied to a Group Footer band, only counts the Data Rows in the individual group.

Count: {Count()}

Figure 01 features an example of the report template for a report with groups set up. Notice, the same expression has been entered in both the Group Footer band and the standard Footer band. Fig. 02 – Group Footers in a Rendered Report
