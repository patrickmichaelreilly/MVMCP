2D Machining Tools

The 2D Machining Tools tab contains several other sub-tabs:


  1. Part Information – This includes information on the part and also allows you to place pods and make single part g-code for point-to-point machines.
  2. Vertical Drilling Tools – Contains a list of available vertical drills based on your machine tool file. It also contains tools for drawing additional vertical drilling operations.
  3. Horizontal Drilling Tools – Contains a list of available horizontal drills based on your machine tool file (for machinery that can drill on part edges). It also contains tools for drawing additional drilling operations.
  4. Routing Tools – Contains a list of available routing tools based on your machine tool file. It also contains tools for drawing additional routing operations.
  5. Route Editing Tools – Contains tools for modifying routes and polylines.
  6. Sawing Tools – Contains a list of available saw tools based on your machine tool file. It also contains tools for drawing additional saw operations.
  7. Hardware – It contains a list of hardware that can be added to a part. Any associative machining assigned to the hardware will be applied to the part.

The tools listed in the Vertical Drilling, Horizontal Drilling, Routing and Sawing Tools sub-tabs are reflective of the tools in your Primary Toolfile, selected in:

Toolbox Setup > Options > Toolfiles.
