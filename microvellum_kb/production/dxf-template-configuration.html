DXF Template Configuration

A DXF Template file has been added for DXF files created during work order processing that includes the Dim Styles established in Options as well as a new viewport.  Now, when DXFs are created the Dim Styles and viewport are pulled from this template, as well as any other modifications made by the end-user, and copied into the individual Part DXFs.

The DXF Template file will be created the first time you process a work order with the "Create DXF Files" checkbox selected. 

The newly created DXF_Template.dxf is located in the Microvellum Data folder by default (the path to the DXF Template can be changed in Options/Machining). 
NOTE: The DXF Template file is used in place of the DXF Settings.  The DXF Settings in Options/Machining will not be accessible as long as there is an existing template file in the "Path to DXF Template" file location.  To access DXF Settings, delete, move, or rename the DXF_Template.dxf file.

Open the DXF Template to modify it by zooming in/out, dragging to change position, changing the dim styles, changing the text.  The center of the template is marked with a target.  The bottom left corner of the parts will be at this location.
Entities cannot be added to the DXF template at this time. 
  IMPORTANT :  When modifications are made to the DXF Template, it MUST be saved as file type “AutoCAD R12/LT2 DXF (*. dxf ).”
