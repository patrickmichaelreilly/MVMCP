Adding a Project Name to a Part Label Scrap Name Field

You may want to add a project name to a part label that contains fields displaying the scrap material names. If this is done using a current default Microvellum part label as a template, the process is quite easy, as shown by the screencast attached.

Here is a general outline of the process:

  • Open the label in the Report Designer.
  • Verify that the necessary database relations exist in the dataset (as shown in the screencast).
  • Double click to open the desired text component.
  • Position the cursor at the appropriate point, and if working with data related to the OptimizationResults data, insert the field definition:       
    OptimizationResults.Parts.Products.Projects.Name
  • Add the ampersand characters and empty string, as shown on the screencast, to make sure there is a space between the project name and the scrap name.
  • Preview the report to make sure there are no errors that will later prevent the label from opening.
  • Save and close the modified report.
