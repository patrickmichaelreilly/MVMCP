Understanding the Product Viewer

The Product Viewer contains a list of all the products that are contained in the current (open) room. It is accessible by clicking on the Product Viewer tab within Toolbox.

Like the Product List, the products displayed in the Product Viewer are color-coded. Products displayed in blue text are products that have been drawn in the current room. Products that appear in the green text have not been drawn in the current room.

By selecting a product and then right-clicking will open the following tool menu. 

  1. Draw Selected Product – Expanded, this menu contains the tools for drawing products and walls.
  2. Show Parts List – Allows you to access the parts list for the selected product. From the parts list, you can draw parts in 2D and access the 2D Machining Tools.
  3. Show Subassembly List – Allows you to view a list of the active subassemblies within a product. For most of the library products, this will need to be viewed before using the “Show Parts List” tool.
  4. Add New Empty Product – Allows you to add a new product with an empty spreadsheet.
  5. Delete Selected Products – Allows you to delete products from the room. The products will be removed from the list and drawing.
  6. Edit Design Data – Opens the Edit Design Data interface for the selected product.
  7. Rebuild Subassemblies from Library – Allows you to rebuild the selected project or unique subassemblies and restore them from the library subassemblies.
  8. Refresh – Refreshes the Product List.

Expanding the “Draw Selected Product” list will display the following menu.


  1. Fill Wall with Products – Allows you to fill a wall with the selected products.
  2. Place Product on Wall – Allows you to place the selected product(s) with a selected anchor point.
  3. Place Product Next to Another Product – Allows you to draw the selected product(s) next to another product.
  4. Place Product at Selected Point – Allows you to place a product at a selected point in the drawing.
  5. Redraw Selected Products from Database – Allows you to redraw the selected products using the product information stored in the database. This tool can be used to restore a room/drawing (3D products only) if it was not saved or in the event of a crash with an unsuccessful backup being created.
  6. Redraw Walls from Database – Allows you to redraw the walls using the information stored in the database. This tool can also be used to restore a room/drawing.
