Master Bathroom Partition Overview

The Master Bathroom Partition

The Master Bathroom Partition product is an array design for toilet and shower bay applications. This includes the proper interior width and depth set for the partition spacing and their function. Each bay has the ability to become a toilet opening or a shower opening, factoring in required options such as splash panels and shower seats. Hardware for partition legs, hooks, hinges and door locks are available and customizable within this product. 

There exist two calculators within this product, the first calculator (Internal Opening Calculator) is used for the interior width required of each bay opening. The second calculator (Front Opening Calculator) is used to find the required opening width for the bay doors. Width alterations in the Front Opening Calculator will control both the Door width and the Fascia widths.
