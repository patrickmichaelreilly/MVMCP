The Closet/Wardrobe Expansion Overview

About the Expansion     

The Microvellum Closet/Wardrobe Expansion contains a wide collection of closet/wardrobe products, inserts, hardware, and customization options to complement the Foundation Library. This article will provide you with an overview of the Closet/Wardrobe Expansion and contains a listing of products, subassembly inserts, and options included.

System 32

An option for a 32mm Euro system style of line bore is included within products found in the Foundation Library Closet Expansion. This standardized system uses holes spaced in 32mm increments for mounting hinges, drawers, and all the necessary hardware needed to assemble a closet or cabinet.

When using System 32, several rules & variables need to be set up & followed. If any of the following items are not set up correctly, issues may occur within your closet products and components.

The line bore is the array of holes that come predrilled on each part. The following rules apply to all line bores:

  1. Holes are 5mm in diameter.
  2. Holes locate all hardware (horizontally and vertically).
  3. Holes are always 32mm apart center to center.
  4. Holes are always 9.5mm from center of hole to edge of part.
  5. Hole drilling determines part size.
  6. The centerline of the part lines up with the centers of the holes used.
  7. The Global variable “Front Shelf Hole to Edge” needs to be set to 37mm to match up with the hinge plate setback from the front edge.
  8. The Global variable “Cam Distance to Front Edge” needs to match the “Front Shelf Hole to Edge”.
  9. The Global variable “Cam Distance to Back Edge” needs to match the “Rear Shelf Hole to Edge”.
  10. The Global variable “Default Closet Toe Kick Height” needs to be in increments of 32mm.
  11. The Height of the closet is determined from the hole count and the hole spacing (32mm). Reference the prompt “Required Height” to match the Height of the product.
  12. The Height of drawer fronts are determined by the “Drawer Hole Count” prompts.
  13. The intended construction method to use is “Cams”.
