Microvellum Flatshot Token (Overview)

As of release 15.6, the speed of processing when using the Microvellum Flatshot token has been significantly increased for some types of products.

More importantly, the token can also display hardware as part of the Microvellum Flatshot token. This is triggered by a new fifth option for parameter 2.

To display hardware in the flatshot image produced, set Parameter 2, Option 5 to a value of “1.” This option is separated from other parameter 2 options with a semi-colon character.
