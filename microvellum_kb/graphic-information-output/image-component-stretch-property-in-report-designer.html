Image Component Stretch Property in Report Designer

The articles in this section of the Knowledge Base involve working with Images for display in your reports. To learn more about Images, and how to access the Image component within the Report Designer, see Loading Images in Report Designer. This article provides an overview of the Image Stretch Property.

Occasionally an Image component is bigger than the actual image meant to fill it. This is when the Stretch property can be used to improve the look of your report.

The Stretch property is located in the properties of an Image component.


Fig. 01 – Stretch Property Location

The Stretch Property is set to False by default.

Fig. 02 – Unapplied Stretch Property and Applied Stretch Property

After setting the Stretch property to true, the image fills all free space of the component.

Stretching an image may break its proportions.

To stretch an image without breaking its proportions, set the Aspect Ratio property to True.


Fig. 03 – Aspect Ratio Property

The Aspect Ratio Property is False by default.

Fig. 04 – Aspect Ratio Example

The Aspect Ratio property has no effect if stretching is not enabled.
