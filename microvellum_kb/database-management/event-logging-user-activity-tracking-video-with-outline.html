Event Logging & User Activity Tracking (Video with Outline)

We have made the following enhancements to Toolbox. Please see the attached video at the time markers indicated for more information.

User Activity Tracking  (time marker 0:10)

To facilitate communication between teams at your company and introduce necessary accountability, we are introducing User Activity Tracking.

This will be especially helpful to senior engineers or company management personnel. They will be able to grant user permissions for specific activities to certain individuals if login is enabled.

Future releases will support tool file and processing station modifications.

Event Logging  (time marker 4:17)

In order to speed-up your use of the software by eliminating the need to click through many unnecessary message boxes, and with the goal of improving our diagnostic tools, we are introducing Event Logging.  The video below provides an overview of the Event Log Interface.  For information on managing your Event Log storage, see Event Log Properties.

Prior to Toolbox build 21.1.0827.641, the EventLogs table in the database was so large for some customers that when they attempted to open the Event Logs dialog, load times were excessive.  Now when opening the Event Logs dialog, the database is no longer queried. To see events, filters need to be set and “Apply Filters” selected. 
Add a date filter for “All Time” to display all your logs.
