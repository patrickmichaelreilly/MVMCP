Microvellum Data Backup Tips

Here are a couple steps you can take to backup your Toolbox data prior to updating your software:

Information on System used in sample paths:

Name of Library configuration:

Advanced_Frameless_Library_English_v036_1

Database Type:
SQLCE

Files To Be Backed Up:

Microvellum Data Folder

Backup of Microvellum Data:

  • Find your path to your most recently used "Microvellum Data" by clicking on "Help" > "Browse to Microvellum Data."
  • Go "Up" one level in the folder structure and copy the entire Microvellum Data Folder (e.g., C:\Users\%userprofile%\AppData\Local\Microvellum\Common Microvellum Data\Advanced_Frameless_Library_English_v036_1).
  • Paste the "Microvellum Data" folder into a backup folder.
