How to Setup a Local Sandbox Environment

This document explains the steps to configure a local sandbox. For additional information about what a “sandbox” is and when to use one click here. Before creating a sandbox environment, we strongly recommend the user back up their production data. 

Steps to Set Up a Local Sandbox

  1. Open Toolbox in a “local” configuration (data not connected to an external server).
  2. At the top of the main Toolbox palette hover, over “Help” and select "Browse to Factory Data."      
  3. Navigate up one folder by selecting the “Up” arrow.

  4. Copy the folder named "Microvellum Data" or the folder containing the Factory Database, and Work Orders. Name the copied folder “Sandbox Local.”                 
  5. Restart Toolbox.
  6. Select “Manage Configurations” within the “Login Screen” (the first screen on startup).
  7. Select and Copy the desired configuration and name it “Sandbox Local."

  8. Go back to the main Login Screen and select Sandbox Local (from the drop-down) as the “Current Configuration."
  9. Select "Toolbox Setup" -> "Options."
  10. Select the “Utilities” tab -> “Change Path to Microvellum Data.”
  11. In the “Browse For Folder” window, select the “Sandbox Local” folder (created in Step 3). 

The local sandbox configuration is now complete and ready for testing
