Installing Gannomat MDB Writer

This tutorial demonstrates the steps recommended for installing the Gannomat MDB Writer. 

Installation Steps 

  1. Close any open Microvellum products.
  2. Download the zipped Gannomat MDB Writer package attached to this article. 
  3. Unblock the zipped installation folder by right-clicking the folder and selecting “Properties”. In the “General” tab, you may see a “security” section near the bottom. Select the “Unblock” checkbox.
  4. Extract the contents of the zipped folder by right-clicking it and selecting “Extract all”. This file can be extracted to any location on your local machine.
  5. Open the unzipped folder, copy the folder named “DataBase Writer x64” and the .bat file named “DatabaseInstall”.
  6. Open Windows File Explorer and navigate to “C:\ProgramData\Microvellum\”.
    1. If you cannot locate the “ProgramData” folder, you may need to click File > Change folder and search options > View tab > and enable Show hidden files, folders, and drives then “Apply to Folders” and OK to exit.

     


  7. Paste the “DataBase Writer x64” folder and the “DatabaseInstall” .bat file into this location. 
  8. Right-click the .bat file named “DatabaseInstall” and select “Run as administrator”.
