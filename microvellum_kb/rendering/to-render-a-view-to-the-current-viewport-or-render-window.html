To Render a View to the Current Viewport or Render Window

Render to the Current Viewport

  1. Display a 3D view of your model. 
  2. Click Visualize tab > Render panel. 
  3. On the Render panel, click the Render Presets drop-down list and select the render preset to set current. 
  4. Click the Render In drop-down list and select Viewport. 
  5. Click Render to Size. 
  6. Optionally, save the rendering to an image file with the SAVEIMG command. 

Render to the Render Window

  1. Display a 3D view of your model. 
  2. Click Visualize tab > Render panel. 
  3. On the Render panel, click the Render Presets drop-down list and select the render preset to set current. 
  4. Click the Render In drop-down list and select Render in Window. 
  5. Click the drop-down menu on the Render to Size button and choose an image size to render. 
  6. Click Render to Size. 
  7. In the Render window, review the rendering. 
  8. Optionally, save the rendering to an image file.
