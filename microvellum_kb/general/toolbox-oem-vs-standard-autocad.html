Toolbox OEM vs Standard AutoCAD

Although most features of Toolbox OEM are exactly the same as their counterparts in AutoCAD, there are several differences. In addition, Toolbox OEM defines a few native commands and system variables that do not exist in AutoCAD.

Differences in Features and Utilities:

Toolbox OEM does not support the following features:
  1. Autodesk Apps Store
  2. Embedded VBA macros
  3. Express Tools
  4. InfoCenter
  5. Geographic location information with online maps
  6. Online Share
  7. Profiles tab of the Options dialog box
  8. Shared views
  9. Save to/Open from AutoCAD Web & Mobile
  10. Drawing History
  11. Recent blocks from the Blocks palette
  12. VLX files compiled with the Visual Lisp IDE or MAKELISPAPP command

Unsupported System Variables:

Toolbox OEM can use most of the Standard AutoCAD system variables; however, the following system variables are not supported:
  1. ACADVER
  2. ACADPREFIX
  3. LOGINNAME
  4. MENUCNTL
  5. PDFSHXLAYER
  6. PDFSHXTHRESHOLD
  7. POPUPS
  8. SCREENBOXES
  9. SCREENMODE
  10. SECURELOAD
  11. SECUREREMOTEACCESS
