About Microvellum (Legacy)

The following information applies to Microvellum software builds prior to build 23.1.1205.642.

The About Microvellum User Interface displays information and links related to your Microvellum software, company account details and other information and functions.

To open the About Microvellum UI click on the Help drop-down menu in the Microvellum Toolbar.

Select " About Microvellum" to open the interface. Each component of the interface is numbered in the reference image and a description is given below.

  1. Build Number - The current software Build Number is displayed here. To learn more about the Toolbox Build Number, see the knowledge base article Microvellum Software Versioning Schema.
  2. Update Available - When an update to your Toolbox Software becomes available, the update link will be displayed here.  Otherwise, "No Update Available" is displayed.
  3. EULA - Click the EULA link to view the Microvellum Software End-User License Agreement.
  4. Update Serial Number - Use the Update Serial Number link only if you need to clear your current serial number and input a new valid replacement serial number.

  5. Start Support Request - Click the Start Support Request link to create a new support ticket.  See How to Create a Support Ticket through the Microvellum About/Support Portal UI for more details. 
  6. Copy Product Information - The Copy Product Information link will copy the information listed in the About Microvellum UI to the Windows clipboard. 
  7. Serial Number Expiration - Your Microvellum Product Serial Number is valid until the date displayed. 
  8. Serial Number - Your Microvellum Product Serial Number is displayed here. 
  9. Request Code - Your Microvellum Request Code is displayed here.  This code is used by Microvellum support to authenticate a product license by phone. 
  10. Software Maintenance Expiration - Your Software Maintenance Expiration date is displayed here. 
  11. Account ID - Your Microvellum Account ID is displayed here.
  12. Account Manager Details - Your Account Manager Details, including name, phone number, and e-mail address are displayed here.
  13. Software Features - A list of software features (or modules) is displayed on the right side of the UI.  The activation status of each software feature is shown as either "Active" or "Request Trial".  Click the "Request Trial" link and fill out the form to send a request to Microvellum to try a new software feature.


  14. Microvellum Knowledge Network - Click the link to visit the Microvellum Knowledge Network.
