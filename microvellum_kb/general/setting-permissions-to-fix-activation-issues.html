Setting Permissions to Fix Activation Issues

Issue:

A user attempts to enter a new serial number, but the software does not allow it, or the activation is not maintained after a Toolbox or computer restart.

Resolution:

Use the procedure below to clear and reset the activation.
  1. Open Windows File Explorer and navigate to “C:\ProgramData\.”
    1. If you cannot locate the folder, you may need to click File > Change folder and search options > View tab > and enable Show hidden files, folders, and drives then “Apply to Folders” and OK to exit.

      2. Fig. 01

  2. Right-Click on the Microvellum folder and select Properties.

  3. Select the Security tab > select the desired user > click the Edit button. Select the desired user again and check the checkbox for Full Control in the ‘Allow’ column. Click OK.



  4. Select the Advanced button in the Microvellum Properties window > select the desired user > select the Change Permissions button > choose Save Changes.
  5. Select the “Replace all child object permissions with inheritable permissions from this object” checkbox > select OK.
