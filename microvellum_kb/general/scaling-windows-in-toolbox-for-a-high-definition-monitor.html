Scaling Windows in Toolbox for a High Definition Monitor

Some users have reported scaling issues with windows inside of Toolbox when they are working on an HD monitor.


Fig. 01 - Example Scaling Issue

There is a quick fix for scaling issues like this demonstrated in Figure 02. 

Fig. 02 - Scaling for HD Monitors Fix
  1. Right-click Toolbox Icon. 
  2. Select Properties. 
  3. Select the Compatibility tab. 
  4. Select Change High DPI Settings. 
  5. Check Override High DPI Scaling Behavior. 
  6. Select the "Scaling Performed by" drop down > Select System (Enhanced)
  7. Select OK > Select OK for changes to be saved. 
If problems persist, ensure the size percentage in the Scale and Layout settings of your work machine is set to 100%.
