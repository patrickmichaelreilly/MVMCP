How to Create and Organize Rooms in Toolbox

Creating and Organizing Rooms  

Once a project has been created or opened, you can create and organize the rooms for the project.

To access the Rooms Tree, click on the Project tab and then click on New Room icon  or Open Room icon .


To edit the Rooms Tree, select a category and right-click. This will display the Rooms menu. When a category has been selected, the following options will be available:


  1. Add New Category allows you to add a new category or sub-category under the selected parent category.
  2. Rename Selected Category allows you to rename the selected category.
  3. Delete Selected Category allows you to delete a category.
  4. Add New Room allows you to add a new room under the selected parent category.

As with naming projects, it is also important to create a standard naming convention for rooms within a project. An example would be something like:

<Sheet #> - <Room Name> <Room #>

1 - Reception 101

When a room has been selected, the following options will be available:


  1. Copy Room - Allows you to copy the selected room.
  2. Rename Selected Room - Allows you to rename the selected room.
  3. Delete Selected Room - Allows you to delete the selected room.
  4. Move Selection to Another Category - Allows you to move the selected room to another category.
  5. Room Properties - Allows you to access the Room Properties window – used to populate attributes within blocks (i.e., finish schedules).

Setting up the Room Properties (Inside of a Project)

After a room has been opened, the Room Properties button will be visible and active. The Room Properties allows you to set up information that is particular to the room. By default, these values populate a table in the Spec sheet of the Microvellum drawing templates. Most of these variables have been set to look at specific variables.


To auto-populate, these values, right-click on a field, and select Edit Room Level Property.


In the next window, click on the Specification Group Name to select which specification group the variables will be referencing.


After selecting the desired specification group and closing out of the Edit Room Level Property window, right-click again and select Update Database. The Room Properties fields will now be populated.
