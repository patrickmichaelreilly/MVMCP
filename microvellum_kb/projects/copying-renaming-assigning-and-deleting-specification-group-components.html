Copying, Renaming, Assigning, and Deleting Specification Group Components

This article summarizes working with Specification Group Components.

For a summary of the Library Specifications Groups Interface, see:
Understanding the Project Specifications Groups Interface

For a summary of the Project Wizard, Global Variables, Material Files, and Door Wizard Interfaces, see:
Understanding the Project Wizard, Global Variables, Material Files, and Door Wizard Interfaces

For working with Specification Group Collections, see:
Copying, Renaming, Assigning, and Deleting Specification Group Collections
To copy a component, you will need to select the component, and click the Copy icon  (or right-click and choose Copy (Make New) Component. Give the component a new name. It is recommended that you create a standard naming convention for your components for better organization.

Here is an example of a naming convention for a plastic laminate scenario:

PL<#> <Manufacturer Abbr> <Color #> <Color Name>

PL1 WA 10776-60 Kensington Maple

To rename a component, you will need to select the component and click the Rename icon  (or right-click and select Rename Selected Component).

To assign a component to a collection, you must select the component in the right section as well as in the left section under the particular collection you want it assigned. To complete the assignment, click the arrow between the left and right sections.

To delete a component, you will need to select an unused component (orange text) and click the Delete icon  (or right-click and choose Delete Unused Component).
