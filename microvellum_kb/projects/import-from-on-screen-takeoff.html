Import from On-Screen Takeoff

Microvellum offers the ability to import an On-Screen Takeoff (CSV) file to create projects, rooms, products, and parts. 

On-Screen Takeoff

Once a project has been exported from On-Screen Takeoff, it can be imported into Toolbox. 
The CSV file being imported MUST be located in the same folder location as the .osm (On-Screen Mapping) file. 
To import an On-Screen Takeoff file, select Toolbox Setup > Import from On-Screen Takeoff. 

Fig. 01 - Import from On-Screen Takeoff

This opens the Import OnScreen Data window where the CSV file can be selected for import. 

Fig. 02 - Import OnScreen Data Window

  1. Select the CSV file to be imported. 
  2. Select Append to Existing Project to add the information to an existing project as opposed to creating a completely new project. 
  3. The Projects to choose from for appending a file to. 
  4. Enter a New Job Name when an Existing Project is not being appended to. 
  5. The Specification Group to be appended to. 
  6. Select whether Linear Feet is to be rounded up or down. 
  7. Select Edit Mapping file to open the OnScreen Mapping window. 
    1. Product Mapping

      2. Fig. 03 - Product Mapping Tab

    2. Default Prompt Maps

      3. Fig. 04 - Default Prompt Maps Tab

The CSV and OSM files are created using On-Screen Takeoff. For more information regarding formatting, see the On-Screen Takeoff KB.
